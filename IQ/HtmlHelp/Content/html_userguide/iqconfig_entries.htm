IQconfig Entries

The IQconfig entry is the basic building block of the IQF. It contains all the information needed to describe itself to the Aspen IQ application.

Contents of an entry

The following list describes the contents of an entry.

  • Name
    A unique text string identifying the entry to the Aspen IQ application.

  • Description
     A text string which provides a brief summary of the entry's function.

  • Type
    The data type determines how the value is to be interpreted.

  • Keyword
    Describes whether the data for this entry resides internally to the Aspen IQ application or externally (in a database point for example), and whether it allows read-only, write-only or both types of access. IQconfig presents you with a drop down list of the valid keywords for each standard entry.

  • Value
    The default value to use when first loading the Aspen IQ application. The value can be a number or text.

  • Entity
    Contains all information needed to locate the value in a database or DCS system. The entity is only used if the keyword warrants it. You can define constant or local values in IQconfig without having to store the data in tag names. The entity itself is comprised of a number of items:

    • Cim-IO Device (optional)
      This information is only needed if Cim-IO is the interface being used. Cim-IO is AspenTechâ€™s real-time process database interface platform. Aspen IQ uses it to communicate with many popular DCS and database systems.

    • Unit # (optional)
      This information is only needed if Cim-IO is the interface being used. You can enter a unit number.

    • Source
      Another type of keyword describing the mechanism for reading or writing the value.

    • Tagname
      A native text string that allows the point to be accessed via the interface.

      Note:  Not every entry requires an entity.

Working with entries

To see more information about an entry, highlight the entry with a right mouse click. Press the F1 key to bring up a help topic window that displays information about the entry from the Aspen IQ Entry Dictionary.

You can choose which entries for a particular folder to view (All, Required, or Optional). Refer to Viewing entries for more information.
