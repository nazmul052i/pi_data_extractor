Unique Application Names Requirement for Aspen Watch Performance Monitor

For online applications to be available for monitoring and use with Aspen Production Control Web Server and Aspen Watch Performance Monitor, the applications must be uniquely named.

Aspen Production Control Web Server

If different types of applications are identically named and hosted on the same online applications server (for example, an Aspen DMC3 Builder application and an Aspen IQ application both named "ABC"), then only one of the applications will appear in Aspen Production Control Web Server pages.

This is a limitation of the web repository, which uniquely identifies applications by host and name. Therefore, only the first application reported is displayed, and the second one is ignored.

Aspen Watch Performance Monitor

Applications monitored by Aspen Watch must be uniquely named in order to be included in the Aspen Watch database and have historical data collected.

This requirement applies to all applications, even if they originate from different development platforms (for example, from Aspen DMC3 Builder or Aspen IQ), or they operate from different online applications servers.
