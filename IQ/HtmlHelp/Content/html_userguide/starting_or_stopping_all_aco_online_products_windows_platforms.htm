Starting or Stopping Online Aspen IQ Applications

Use Aspen APC Manage to start or stop all ACO online applications and components, such as Aspen Inferential Qualities (Aspen IQ), Aspen Nonlinear Controller (Aspen Apollo), Aspen DMCplus, and the ACO Message Server.

Important:  The commands for either global startup or shutdown of all ACO online products should be used with appropriate caution.

To shutdown ACO Base:

  1. Access the Aspen APC Manage program:

    • From the Windows™ Start menu, click Programs | AspenTech | Aspen Manufacturing Suite | APC Online | APCmanage.

      Tip:  From the Start screen of Windows, press the Windows key + Q, and search for "APCmanage".

      The Aspen APC Manage program starts.

  2. From the main menu of Aspen APC Manage, click Tools | Shutdown Advanced Control.

To startup ACO Base:

  1. Access the Aspen APC Manage program:

    • From the Windows™ Start menu, click Programs | AspenTech | Aspen Manufacturing Suite | APC Online | APCmanage.

      The Aspen APC Manage program starts.

  2. From the main menu of Aspen APC Manage, click Tools | Startup Advanced Control.
