Using an IQmodel Model for Prediction

When using an IQmodel Model in your prediction module, you must complete the following steps.

To configure a prediction module to use an IQmodel model:

  1. After selecting to use an IQmodel Model and clicking OK to close the Model Type dialog, the IQmodel Model dialog appears, with three tabs on it: Model, Inputs, and Outputs.

  2. On the Model tab, locate the model file you want to use. Click the Browse button to display an Open dialog. Navigate to the desired path (drive and directory) where the file is stored. Only files with an extension of .iqr are shown. Double-click the desired file name. The name appears in the Model File field and all model variables appear in the Variable list.

  3. Enter values for the Output Filters, Tau 1, Tau 2, and Deadtime.

  4. Click the Inputs tab. For each input, you can dynamically shift the inputs with a dead-time and two lags (Tau 1 and Tau 2).

  5. An input can also be the result of another calculation, which you specify using the Input Calculation field. You can call up the Function Wizard to help you create a calculation; click the F(x) button.

    As new inputs are entered into the calculation specification, they appear in the Calculation Variables: Values and DCS connections list.

  6. Assign the appropriate source and tagname to the individual inputs in the Calculation Variables: Values and DCS connections list. The CIM-IO Device and Unit # are not required if they are the same as the default device specified in the Config section.

  7. You can create both dynamic and steady-state model outputs. Any dynamic models that are specified in IQmodel will automatically be specified to the unbiased dynamic prediction, UAZ, parameter and the unbiased steady-state prediction will be sent to the UPR parameter. These values can be written to the database by assigning each a tag name. Click the Outputs tab and set the Cim-IO Device, Unit #, Source, and Tagname, as appropriate, for the Outputs and Results.

  8. Click OK when done. The IQ is created.

    Clicking the Change Type button can change the model type. The UAZ and UPR parameters for the inferred quality are automatically added to the data collection lists.
