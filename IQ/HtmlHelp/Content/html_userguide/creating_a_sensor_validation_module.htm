Sensor Validation Module

To create a sensor validation module:

  1. When creating a new IQ, select the sensor validation check box from the Add Inferential Quality dialog box.

  2. You will then see the Sensor Validation dialog box.

  3. Select your Sensor Validation Model file. (*.svr) by clicking on the button next to the Model File field.

  4. Your Sensor validation model file is now loaded. See Sensor Validation for details on configuring your sensor validation model.
