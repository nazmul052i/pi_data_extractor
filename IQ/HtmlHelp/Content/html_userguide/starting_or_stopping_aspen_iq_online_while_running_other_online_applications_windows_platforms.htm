Starting or Stopping Online Aspen IQ Applications

See Also

Online Applications Management Overview

Use Aspen APC Manage to start or stop online Aspen IQ applications, without stopping and restarting other online applications such as Aspen DMCplus controllers or Aspen Nonlinear Controller (Aspen Apollo) applications.

To stop Aspen IQ applications (online):

  1. Access the Aspen APC Manage program:

    • From the Windows™ Start menu, click Programs | AspenTech | Aspen Manufacturing Suite | APC Online | APCmanage.

    Tip:  From the Start screen of Windows 8 or Windows Server 2012 or later, press the Windows key + Q, and search for "APCmanage".

    The Aspen APC Manage program starts.

  2. In Aspen APC Manage, do either of the following:

    • From the main menu of Aspen APC Manage, click Tools | Shutdown Aspen IQ.

      -or-

    • In Aspen APC Manage, select the IQ Applications tab.

    In the IQ Applications tab, select all applications that are operating online, and then click the Stop button.

    As a result, this stops:

    • Any running Aspen IQ applications

To start Aspen IQ applications (online):

  1. Access the Aspen APC Manage program:

    • From the Windows™ Start menu, click Programs | AspenTech | Aspen Manufacturing Suite | APC Online | APCmanage.

    The Aspen APC Manage program starts.

  2. In Aspen APC Manage, do either of the following:

    • From the main menu of Aspen APC Manage, click Tools | Startup Aspen IQ.

      -or-

    • In Aspen APC Manage, select the IQ Applications tab.

    In the IQ Applications tab, select all applications that are operating online, and then click the Start button.

    As a result, this starts:

    • Any Aspen IQ applications marked for autostart
