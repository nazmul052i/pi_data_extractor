Get History Record Information

Choose this option to display a list of tagnames (also referred to as labels) stored in the history file. This list contains:

  • History tagname

  • History tagname index

  • Time of first (oldest) data record for this tag

  • Time of last (latest) data record for this tag

  • Number of data records

  • Maximum number of data records

At the beginning of each history tag is a selection flag. An asterisk ‘*’ indicates that the tag will be included for extraction when the command (or menu option) to generate a CLC or CSV format output file is selected.

The list of tags selected for extraction can be changed using the select and deselect commands, or the corresponding menu options.

Select history records for extraction or clear history records from the extract list. When a history file is opened, all the tags in the history file are selected for extraction by default.

Format arguments for the recinfo command

You can specify a different format for the list in the optional argument for the recinfo command. The following codes are used to represent specific pieces of information:

  • %N – History label

  • %G – History DCS tag name

  • %D – History description

  • %U – History engineering units

  • %E – Percent Tolerance

  • %R – Percent Good Required

  • %# – History tag index

  • %F – First history record time

  • %L – Last history record time

  • %Q – Update Frequency

  • %n – Number of History records

  • %m – Maximum history records

  • %* – Selection indicator (*)

  • \n – End of line (carriage return)

  • \t – Tab

Any other text will be reproduced as is.

Note:  The above codes are case sensitive. For example, %f will not work.

Tip:  You will typically want to end the line using the \n code (for end of line, or carriage return). Otherwise, the list will all be one line.

Default format

The default format as used by the menu option is:

"%*%# %N %F %L %n %m\n"

Note:  The format of the timestamp values (which include %F and %L) can be modified by using the Aspen IQ History Output Options Menu - option [2] Set the date and time format.
