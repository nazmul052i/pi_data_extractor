Aspen IQ Directory Structure

The following chart shows the Aspen IQ software directory structure.

Directory Names and Structure

Description

APCONLINE

 

 

Root directory for Aspen IQ online program and executable files

 

bin

 

Programs and command files

 

inc

 

Include files

 

lib

 

Library files

ONLINEAPPS

 

 

Root directory for hosting deployed online applications

 

app

 

 

 

 

iqapp1*

Directory where user-created model files (MDL, IQR, or AIQ) and configuration files (IQF) are stored for a specific application.

(See Important note below.)

 

 

iqapp2*

Directory where user-created model files (MDL, IQR, or AIQ) and configuration files (IQF) are stored for a specific application.

(See Important note below.)

 

 

iqapp3*

Directory where user-created model files (MDL, IQR, AIQ, or RNN and PB) and configuration files (IQF) are stored for a specific application.

(See Important note below.)

 

cfg

 

Miscellaneous configuration files

 

etc

 

Miscellaneous Aspen IQ log files

 

sys

 

System management directories and associated files

* Important:  You must create and name one subdirectory for each Aspen IQ application that will be used. The name of the directory is the name of the application.
