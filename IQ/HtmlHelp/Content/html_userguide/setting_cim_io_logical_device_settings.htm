Setting Cim-IO Logical Device Settings

After selecting Cim-IO as the Connect protocol, you will see the following fields:

The Default Logical Device field shows the Cim-IO logical device to be assigned as the "(Default)" for the IQF. If you have not set one yet, the field states Undefined.

The Default Unit Number field allows you to enter a default unit number for the default logical device.

The Set as Default button allows you to select which logical device name to be used as the default; this causes Connect to assume this logical device will be used for any entity that uses the "(Default)" device name.

The New button allows you to define a new logical device name. The device name(s) you enter here will then appear in the list of available devices in the entry editor.

The Delete button allows you to remove a logical device name from the list of available ones.

The Logical Device Name list shows all logical device names you have created, along with their Case Sensitivity and Space Sensitivity settings. Click the label above a column to sort the list alphabetically (or in reverse alphabetic order if you click the label again).
