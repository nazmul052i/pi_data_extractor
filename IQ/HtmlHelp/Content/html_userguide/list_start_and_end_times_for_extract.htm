List Start and End Times for Extract

Choose this option to view the currently selected start and end times for data extraction and the available range of timestamp values from the list of selected tags. The following information is displayed:

  • Extraction start time.

  • Extraction end time.

  • Oldest data time for selected records.

  • Latest data time for selected records.

Format arguments for the liststartend command

You can specify a different format for the list in the optional argument for the liststartend command. The following codes are used to represent specific pieces of information:

  • %S – Extraction start time

  • %E – Extraction end time

  • %F – Oldest data time for selected records

  • %L – Latest data time for selected records

  • \n – End of line (carriage return)

  • \t – Tab

Any other text will be reproduced as is.

Note:  The above codes are case sensitive. For example, %f will not work in place of %F.

Tip:  You will typically want to end the line using the \n code (for end of line, or carriage return). Otherwise, the list will all be one line.

Default format

The default format as used by the menu option is:

"Extraction Start time:

%S\n" +

"Extraction End time:

%E\n" +

"Oldest data time for selected records:

%F\n" +

"Latest data time for selected records:

%L\n"

Note:  The format of the timestamp values (%S, %E, %F and %L) printed in this list can be modified using the Aspen IQ History Output Options Menu - option [2] Set the date and time format.
