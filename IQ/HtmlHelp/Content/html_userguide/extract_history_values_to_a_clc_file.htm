Extract History Values to a CLC File

Choose this option to generate a CLC file containing the history data for the selected tags between the specified start and end times. When a history file is opened, all the tags in the history file are selected for extraction. Also, the extraction start and end times are set to include all the data stored in the history file. These can be modified using the menu commands under the Aspen IQ History Output Menu.

The CLC file generated by IQextract can be loaded in IQmodel and DMCplus Model programs.
