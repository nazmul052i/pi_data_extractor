Algebraic Equation-Based Inferential Sensor

Aspen IQ allows you to enter sophisticated algebraic expressions.

The algebraic expression can use functions from a built-in library. This function library contains:

  • Mathematical functions such as @exp for exponentiation and @log for taking logarithms.

  • Logical functions such as @if, @and, @or, etc.

  • Common process variable transformations used in refinery applications such as @PFPCT for petroleum fraction pressure compensated temperature.

The algebraic equation based inferential sensor models are steady state models.

Refer to Using an Algebraic Model for prediction for details.
