Open History File

Choose this option to select a history file. You must select this option prior to selecting other options in order to get any useful results from the other commands.

The IQextract program will prompt you for how you want to specify the history file.

 1. Select history file for an Aspen IQ application

 2. Enter history filename directly

 

Select history file option (1-2) [1]:

Selecting option 1: Select history file for an Aspen IQ application

The default choice of option 1 is the easiest way to access history files used by Aspen IQ applications. The first option (1) will prompt you to enter an Aspen IQ application name, followed by the type of history file. (This is shown in the example below.)

Enter Aspen IQ application name [ ]: iptest1

Select history file type (1-4) [1]:

Selecting option 2: Enter history filename directly

The second choice is useful for accessing history backup files or history files located in user directories.

The filename of the selected history file will be displayed along with the status of the history open command.
