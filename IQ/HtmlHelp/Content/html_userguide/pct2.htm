PCT2

Pressure Compensated Temperature

In control applications, a column temperature is frequently used as an indication of composition, which is only valid if the pressure is constant. By using an equation of state, the measured temperature may be compensated to a reference state. This compensated temperature accounts for pressure changes and is therefore a better indication of composition.

The BIAS is an optional correction term for cases in which the pressure compensated temperature is used as an approximation of a laboratory analysis. If this feature is not desired, it should be set to 0.0. Either natural log or common (base 10) log may be used: an indicator is set to select the desired basis.

The B & C constants used must be consistent with the temperature and pressure units used. Beyond this, there is no restriction on units or type (absolute temperature or other temperature scale). If NON-ABSOLUTE temperatures are used; the value of C in effect will include the conversion constant. The pressure used for determining B & C must be absolute.
