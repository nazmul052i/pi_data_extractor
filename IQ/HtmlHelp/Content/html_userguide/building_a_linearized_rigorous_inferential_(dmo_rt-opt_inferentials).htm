Linearized Rigorous Inferential Sensor (DMO/RT-OPT Inferentials)

See Also

Using a DMCplus Model for prediction

AspenTech can provide a linearized rigorous inferential based upon its closed-loop real-time rigorous modeling and optimization package RT-OPT (also referred to as a DMO inferential).

The approach centers around a validated rigorous, first-principles model for the desired unit (or area of the unit). After a model is built and validated, a sensitivity analysis is performed. The resulting gain matrix is used to provide the basis for the inferential model.

Proprietary techniques are used to generate a linearized model for the desired inferential quality.

This approach is limited to those product qualities that are currently available in the modeling package (primarily used in the Refining industry). Typically, distillation properties including IBP and FBP can be generated with this package. However, models for cold properties such as flash, freeze, cloud, and pour point must be generated using other approaches.

To implement the DMO inferential in the Aspen IQ environment, you can enter the resulting expression provided by the AspenTech engineer as an Algebraic Equation-Based Inferential Sensor.
