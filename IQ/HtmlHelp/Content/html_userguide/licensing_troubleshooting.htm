Licensing Requirements

If Aspen IQconfig or Aspen IQmodel fails due to a licensing problem, it is necessary to correct the licensing problem before the application can be restarted.

On local computers, use the SLM Configuration Wizard to set license options such as broadcast settings, server names, and so forth.

To access the SLM Configuration Wizard and ensure proper settings:

  1. From the Windowsâ„¢ Start menu or Start screen, search for "SLM License Manager", or click Programs | AspenTech | aspenONE SLM License Manager.

  2. From the top, main menu of aspenONE SLM License Manager, click Configure. Then, in the User Account Control dialog, click Yes to allow changes.

    The SLM Configuration Wizard dialog is displayed.

  3. In the Network License Servers pane, ensure a valid, operating license server is added to the table, unless a local key is installed.

    Typically a license server is already added to the table as a result of successful installation of AspenTech software.

    If, however, the license server host has changed, or there are multiple license servers operating at your site, then use the Add Server button to add valid license server descriptions to the table, and use the Remove button to delete invalid license server descriptions from the table.

  4. Expand the Advanced Settings pane to view the options it contains.

  5. In the Advanced Settings pane, ensure the following options are configured:

    • Expiration Reminder (days) = 60
    • Enable Broadcasting = check box cleared
    • Log timezone information = check box selected
    • Ignore local keys = check box cleared typically, unless a local key is installed
    • Search Configured Servers for all available buckets at Runtime = check box cleared
    • Log IP addresses = check box selected

If you are still having licensing problems after setting the license options on the client computer, see your SLM server administrator or consult the Troubleshooting chapter of the Software License Manager (SLM) Installation and Reference Guide for more information.
