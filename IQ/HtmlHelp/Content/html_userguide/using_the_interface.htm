Using the IQconfig Interface

Note:  When using IQconfig, the phrase "click the mouse button" means to press and release the left mouse button. If you must click the right mouse button to perform a function, the instructions indicate "right click."

Using the IQconfig window

The IQconfig window consists of two panes: the tree view (left side) and entry view (right side)

The tree view displays one or more folders, which organize the entries in the IQF. The first folder is the Config folder, which contains all the general entries associated with the IQF. Each IQ you create will have its own folder. Under each IQ will be one or more module folders. A module folder can have other items placed under it.

To change the size of the displayed area for the tree view pane, place the cursor over the frame between the left and right pane. The cursor changes to a left/right arrow pointer. Click and drag the pointer to the right to increase the size of the left pane.

You can display a pop-up menu listing relevant functions by placing the cursor inside the tree view pane and clicking the right mouse button.

The entry view looks similar to a spreadsheet, with rows and columns. In this view, you will either see entries that you must configure, or data collection-related items. Each row contains one entry, while the columns list various parameters for an entry. Some cells in the row are view only, while others can be changed.

Using the tree view

IQconfig uses a Windows Explorer-like interface to present the structure of an IQF and its entries. A graphical tree structure is used (called tree view). This hierarchy uses folders to group items together. Folders can include other folders, or a list of entries. To open a folder, double-click it or click the plus sign next to it.

For example, selecting an IQ folder in the tree view displays a list of all the modules included in it. A module folder can have other folders placed under it (example: the PR folder will have MVAR and CVAR folders under it).

Using the entry view

The cells with gray background are read-only. The values in cells with a white background can be changed. You can only move to cells that can be changed. To move from one cell to another, press Tab. This will move to the next cell that can be changed. The highlight moves from the current field to the next. To move back, hold down Shift then press Tab.

To select a row, click any cell for the row.

To change the size of the displayed area for the right pane, place the cursor over the frame between the left and right pane. The cursor changes to a left/right arrow pointer. Click and drag the pointer to the left to increase the size of the right pane.

The right pane includes horizontal and vertical scrollbars so you can scroll to view areas of the pane not currently visible.

To change the size of a column (and the amount of information you can see for the column), place your cursor on the right edge of the column label. The cursor changes to a left/right arrow pointer. Click and drag the pointer to the left or right to change the size of the column. To automatically size the column to fit the widest text, select the Resize columns to fit data menu item under the View menu.

To change the sort order of the entries, select a column heading to sort by (either name, description, type, keyword, value, or entity) then click the heading. The entries are re-sorted in ascending order (alphabetically and/or numerically, as appropriate) using the selected heading. If you click the heading again, the entries are re-sorted in descending order.

Using menus

  1. Click the menu name to display its contents. Optionally, you can use a keystroke to open the menu. Hold down the Alt key, then press the letter key corresponding to the underlined character in the menu name (example: Alt + f opens the File menu).

  2. With the menu displayed, drag the highlight over the desired menu item and click the mouse button. The function is performed. Optionally, you can use a keystroke to choose the menu item. Hold down the Alt key, then press the letter key corresponding to the underlined character in the menu item (example: with the File menu displayed, pressing c closes the current IQF).

If an item on the menu is dimmed or gray in appearance, that item is not available in the current context.

Some menu items are toggle switches, meaning that when you select them, they turn off or on. For example, the View/Toolbar menu item turns on/off (shows or hides) the Toolbar.

Using menu shortcuts

Some menu items have a corresponding keyboard shortcut, such as File/New (Ctrl + n) and File/Open (Ctrl + o). These shortcuts are listed on the menu next to the item. To use the shortcut, you do not need to open the menu; only press the indicated combination of keys. For example, to create a new CCF, hold down the Ctrl key and press the n key (shortcuts are NOT case sensitive).

Using the popup menu

A popup menu is available when the cursor is placed inside the left pane of the IQconfig window. To display the popup menu, click the right mouse button.

Using Tool bar buttons

To use a Tool bar button, simply click the button. The function is performed.

Selecting items

To select an item, click it. It is highlighted.

Entering text

To enter text into a field, move the cursor over the field and click. A blinking insertion point (vertical bar) appears, indicating you can now type. You can use the Backspace or Del key to remove text. You can also highlight text to type over it. For inserting text, use the left/right arrow key to move the insertion point to a new position in existing text and then type the new text.

Using drop down lists

A drop down list is a field that has a preset list of choices from which you can select an item. It is indicated by a down arrow next to a field. To use a drop down list:

  1. Click the down arrow next to the field to display the list.

  2. Move the highlight down over the desired choice and click the mouse button. The selected item is placed in the field and the list disappears.

Using check boxes

Some fields are checkboxes, meaning you can click in the checkbox to select it. A checkmark appears inside the checkbox to indicate the field is selected. To deselect the checkbox, click in it again and the checkmark disappears.

Using Tab sections

Some dialog boxes have Tab dividers or sections for presenting fields/text. To select a Tab, click it. The corresponding items are displayed.

Open/Save dialog boxes

These dialogs work just like those in other Microsoft Windows applications.

Scrolling

To scroll, either click an arrow at the end of the scrollbar, click the scroll box and drag it to a new position on the scrollbar, or place the cursor over a point in the scrollbar and click the mouse button.
