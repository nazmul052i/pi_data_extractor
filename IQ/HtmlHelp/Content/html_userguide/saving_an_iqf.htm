Saving an IQF

To save an IQF:

  1. Click Save or open the File menu and select Save. A validity check is run. An IQconfig Validation window displays as the check is done. If no errors occur, a message displays stating "Validation Finished. No errors detected" . If errors occur, go to step 3.

  2. Click OK to close the IQconfig Validation window. Your IQF is saved.

  3. If errors occurred, a message states the number of errors that occurred. Click OK to close the message. Another message displays, asking if you want to save even though errors were detected. Click Yes to save or No to cancel the save.

  4. You can review any reported errors in the IQconfig Validation window and correct them in the IQF.

  5. You can also copy error messages from the window and paste them into a word processor to review as necessary. Click to place an insertion point inside the window and drag a highlight over the desired text. Then press Ctrl and the c key at the same time. The text is copied to the Clipboard; you can paste it into a word processor.

  6. To close the IQconfig Validation window, click Close.

To save an IQF named differently than the current one:

  1. Open the File menu and select Save As. The Save as... dialog displays.

  2. Select the desired drive and directory where you want to save the IQF.

  3. Type a new name for the IQF in the File name field.

  4. Click OK. A validity check is run. See the steps described above about saving an IQF for details on handling any detected errors.
