Entry Keyword

The keyword is used to group entries by storage mechanism and I/O mechanism on the value.

Only meaningful keywords for each entry will be allowed when using IQF.

Keywords are:

(None)

Mark this entry as not used.

IQconfig

Reserved keyword for IQconfig-only entries

CONS

Constant value, set once at load-time (not changeable)

LOCAL

Local value in Aspen IQ shared memory (changeable in online views of the IQ application)

RDWRT

Read and write from/to a PCS (Process Control System) tag

READ

Read only from a PCS (Process Control System) tag

WRITE

Write only to a PCS (Process Control System) tag
