Configuring Transforms

To configure the output transformation type:

  1. In the Prediction Module Properties dialog, click the Change Output Transform Type button. The Configure Output Transformation dialog box is displayed.

  2. To view the list of available transforms, click the button for Transform Type. The following list is displayed.

  3. Select the required Transform type and click OK. For more information on each Transform type and the options available see:

    • Natural Log Transform

    • Log Base 10 Transform

    • Modified Natural Log Transform

    • Modified Log Base 10 Transform

    • Piecewise Linear Transform

    • Shift Rate Power Transform

    Note:  The none option is displayed when none of the transform options are selected.
