Initializing Predictions

You can set initialization of predictions in Aspen Production Control Web Server, after you access the Engineering view for a particular IQ. This is equal to entering a value for PREDINIT in IQconfig (0 for Off or 1 for On).

To initialize predictions:

  1. In Aspen Production Control Web Server, access the Engineering view for a particular IQ. The view provides three panels for PREDICTION, LAB, and ANALYZER information.

  2. In the PREDICTION panel, click the Operate link, as illustrated below.

    As a result, the Detail table — displayed below the three panels for PREDICTION, LAB, and ANALYZER — provides access to an "Operate" set of links and information.

  3. In the Detail table, click the Initialize Prediction?blue link text. The dialog box for initializing prediction is displayed.
