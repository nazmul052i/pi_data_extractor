Using IQextract

The utility program IQextract is the primary user interface you will use for reviewing the contents and extracting information from history files generated by the Aspen IQ applications.

You can run IQextract in menu mode or command line mode. In menu mode, IQextract prompts you with lists of functions to perform. The majority of functions require that a history file be specified first. After opening the specified history file, you can review the history file contents such as the last update time, view the number of data records stored, set the start and end times for extraction, and extract data to CLC or CSV format files.

All the menu commands are also accessible directly from the command line. However, because of the sequential nature of the commands, a series of commands starting with an OPEN command (to specify the history file) and ending with QUIT are expected. The list of commands can be read from an ASCII file or entered directly from the terminal.

To see a list of the IQextract command arguments:

  1. Type the following, and then press the ENTER key:

    iq_extract -h

To see a complete description of a specific command argument:

  1. Type the following, and then press the ENTER key (however, do not include the brackets when you type the <command_name>):

    iq_extract -h <command_name>

Running IQextract

Note: To run IQextract, you must be logged in to a privileged account that has the Aspen IQ environment available.

To start IQextract:

  1. Type the following, and then press the ENTER key:

    iq_extract

    As a result, the IQextract main menu is displayed.
