Checking the Validity of an IQF

To perform a validity check on your IQF:

  1. Either click the button or open the Edit menu and select Validity Check.

  2. When the validity check starts, an IQconfig Validation window indicates the progress of the validity check.

    When the check is done, a message is displayed in the IQconfig Validation window indicating how many errors were found. Click the OK button to close the message.

    You can review errors in the Validation Report window. The pushpin button on the window keeps the window on top of the IQconfig window, so you can review any reported errors and correct them in the IQF.

  3. You can also copy error messages from the window and paste them into a word processor to review as necessary:

    • Click to place an insertion point inside the window and drag a highlight over the desired text.

    • Then press Ctrl and the c key at the same time. The text is copied to the Clipboard; you can paste it into a word processor.

  4. To close the Validation Report check window, click the close button.

Note:  A validity check is also automatically run when you save an IQF.
