Creating an ASCII File with Tag Names

To import tag names from an ASCII file, you should use the following format, without column headers. You can use a comma, space, or tab as a delimiter.

Example

""::"T39425.PV":DBVL:,Tray 25 Temp,DEGF,T39425.PV

""::"T36426.PV":DBVL:,Tray 26 Temp,DEGF,T36426.PV

""::"T32427.PV":DBVL:,Tray 27 Temp,DEGF,T32427.PV

""::"T39428.PV":DBVL:,Tray 28 Temp,DEGF,T39428.PV

""::"T34429.PV":DBVL:,Tray 29 Temp,DEGF,T34429.PV

Explanation

For the example shown above, on line 1:

  • ""::"T39425.PV".DBVL

    is the full Cim-IO entity, which includes:

    • ""

      is the Cim-IO device - substitution

    • ::

      is the unit number - substitution

    • "TC39425.PV"

      is the tagname

    • DBVL

      is the source

At the end of line 1:

  • Tray 25 Temp

    is the description

  • DEGF

    is the engineering units

  • T39425.PV

    is the label
