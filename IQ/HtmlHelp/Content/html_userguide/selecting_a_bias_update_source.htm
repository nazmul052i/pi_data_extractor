You can select a bias update source in Aspen Production Control Web Server, after you access the Operations or Engineering view for a particular IQ. This is equal to entering a value for UPDSRC in IQconfig.

You can also view or revise the bias value.

To select a bias update source:

  1. In Aspen Production Control Web Server, access the Operations or Engineering view for a particular IQ. The view provides three panels for PREDICTION, LAB, and ANALYZER information.

  2. In the PREDICTION panel, click the Source blue link text.

    The blue link text shows the current selection for bias update source, and clicking the link displays the Bias Update Source dialog box.

  3. In the Bias Update Source dialog box, select from the following as a bias update source:

    • Manual - Bias can be updated manually.

    • Analyzer - Bias is updated with the analyzer value.

    • Lab - Bias is updated with the lab value.

    • Both Analyzer and Lab - Use both lab and analyzer values to update the bias.

To revise the bias value:

  1. From the PREDICTION panel. click the Bias blue link value.

    The Prediction Bias Used in Online Prediction Calculation dialog box is displayed.

  2. Use the Prediction Bias Used in Online Prediction Calculation dialog box to revise the bias value if necessary.
