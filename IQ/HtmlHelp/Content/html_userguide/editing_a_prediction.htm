Prediction (PR) Module

To edit a prediction module:

  1. In the tree view, click the desired PR (Prediction) folder.

  2. Click the button or open the Edit menu and select Properties. The appropriate Model dialog appears.

  3. Complete the dialog. Refer to the appropriate dialog description, based on the model you selected.

  4. Click OK when done.
