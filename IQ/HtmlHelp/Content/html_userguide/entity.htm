Entry Entity

The entry entity consists of the following:

  • A "source" and "tag name" (regardless of the protocol selected).

  • If using the Cim-IO protocol, a "device" a "unit number" and "format code."

Notes:

  • If the source and tag names are blank, you will see an error during validation. If the device name, unit number, and format code are blank, the default from the Options dialog box is used.
  • You will not see the format code in the Entity column of the entries pane.
