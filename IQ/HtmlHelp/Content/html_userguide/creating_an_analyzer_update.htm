Creating an Analyzer Update Module

See Also

Analyzer Update (AZU) Module

To create an analyzer update module:

  1. In the tree view, click the desired AZU (Analyzer Update) folder.

  2. Click the button or open the Edit menu and select Properties. The Analyzer Update dialog appears.

  3. Select the type of Analyzer Bias Update: Validation Only, Traditional, Scores, or Cusum.

  4. Indicate the Number of Previous Samples Connected to Database (either enter a value or use the up/down arrows to set the number).

  5. Select the New Analyzer Option: Flag, or No DCS. Monitor Changes.

  6. Click OK when done with the Analyzer Update dialog.
