DCS Tag Name Limitations Reference

Aspen Process Control (APC) products have distributed control system (DCS) tag name limitations as noted below.

Aspen DMCplus

  • DCS tag name - 1 to 56 characters.

Aspen DMC3 Builder

  • DCS tag name - 1 to 256 characters.

Aspen Inferential Qualities (Aspen IQ)

  • DCS tag name - 1 to 56 characters.

Aspen Watch Performance Monitor (Aspen Watch)

  • DCS tag name - 1 to 39 for IoGetDef
  • DCS tag name - 1 to 79 for IoLongTagGetDef

Aspen Advanced Control Collect (collect.exe)

  • DCS tag name - 1 to 49 characters
