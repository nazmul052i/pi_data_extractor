Shift Rate Power Transform

Transform Overview

This is a shift, rate, and power transform.

The Shift Rate Power transform implements the following transform:

y = b(x+a)^c

If b and c are < 0, then the equation has the form:

y = b/((x+a)^-c)

The transform statement uses the following form:

SHIFTRATEPOWER shift rate power

where:

  • Shift
    Corresponds to the term a in the equations above. It is a real number that is added to the value to be transformed.

  • Rate
    Corresponds to the term b in the equations above; must have the same sign as power. The shifted, raised value is multiplied by this real number.

  • Power
    Corresponds to the term c in the equations above; must have the same sign as rate. The shifted value is raised to this power (real number).

Notes and example

The input to the transform should be a real number which, when shifted, is greater than or equal to zero. An input of a negative number will cause the transform to fail and will mark that variable or attribute as a bad value.

This example shifts each value up by 1.0, cubes the shifted value, and then divides -0.5 by the result:

SHIFTRATEPOWER 1.0 -0.5 -3.0
