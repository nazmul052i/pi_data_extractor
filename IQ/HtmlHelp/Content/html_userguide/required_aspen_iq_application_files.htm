Aspen IQ Directory Structure

Example Application Builder Files for Learning and Demonstration

For each new Aspen IQ application you want to create, you must create a subdirectory with the same name. This subdirectory must be placed under the ONLINEAPPS\app directory, where all required files for the application will be stored.

You can name the subdirectory any desired name; however, the name will be used as the application name. Typically, the subdirectory name matches the IQF name. You can use Aspen APC Manage to create this subdirectory.

In this subdirectory, you must place the configuration file IQF and all related model files (e.g., MDL, IQR, AIQ, RNN, PB) for the application (if applicable). We also suggest you place all the IQextract output files in this subdirectory.
