Initializing the Lab Bias

You can initialize lab bias in Aspen Production Control Web Server, after you access the Engineering view for a particular IQ. This is equal to entering a value for LBUINIT in IQconfig (0 for Off or 1 for On).

Initializing the lab bias marks all previous lab samples as obsolete and uses the next lab sample to reinitialize the prediction bias so that the prediction matches the lab.

When initializing lab bias, keep the following in mind:

  • The LBUINIT flag stays at one until you reset it to zero or until a new valid lab sample arrives.

  • The LBIASRAW and the LBIASNEW values will be set to zero. This will cause a bump in the prediction if it is using lab updates. The bump is intentional and it is assumed that LBUINIT is changed during a transition when such a disturbance is expected.

  • Once the new lab sample arrives all lab samples taken prior to the setting of LBUINIT will be set permanently to a status of "Bad: Obsolete Sample."

  • If the new lab sample passes validation then the PREDBIAS will be set such that the biased steady state prediction equals the lab value. This will occur even if the engineer has designated "multiple samples are required for bias updates" because this is an initialization operation. The LBUINIT flag is then set back to zero.

  • If the new lab sample does not pass validation then the PREDBIAS will not be updated and the LBUINIT flag will stay at one so that the next lab sample will initialize the PREDBIAS. If the lab fails validation, a new lab sample must be collected and pass validation before the flag is set back to zero.

  • Subsequent lab updates will follow the normal IQ algorithms. For example, if lab updates are configured to require three lab samples then the next lab sample, the second sample since initialization, will not trigger an update.

To set lab bias initialization:

  1. In Aspen Production Control Web Server, access the Engineering view for a particular IQ. The view provides three panels for PREDICTION, LAB, and ANALYZER information.

  2. In the LAB panel, click the Operate link, as illustrated below.

    As a result, the Detail table — displayed below the three panels for PREDICTION, LAB, and ANALYZER — provides access to an "Operate" set of links and information.

  3. In the Detail table, click the Initialize Lab Bias?blue link text. The dialog box for initializing lab bias is displayed.
