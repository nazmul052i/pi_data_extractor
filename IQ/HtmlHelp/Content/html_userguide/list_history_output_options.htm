List History Output Options

Choose this option to display the current setting for the history output options. The following information is displayed.

  • Output time format

  • Maximum tags per page

  • Maximum value width

  • Status column print option

  • Print –9999 on bad status option

Format arguments for the listoutopt command

You can specify a different format for the list in the optional argument for the listoutopt command. The following codes are used to represent specific pieces of information:

  • %T – Output time format

  • %W – Maximum tags per page

  • %V – Maximum value width

  • %C – Printing status column

  • %B – Printing -9999 on bad status

  • \n – End of line (carriage return)

  • \t – Tab

Any other text will be reproduced as is.

Note:  The above codes are case sensitive. For example, %w will not work in place of %W.

Tip:  You will typically want to end the line using the \n code (for end of line, or carriage return). Otherwise, the list will all be one line.

Default format

The default format as used by the menu option is:

"Output time format:

%T\n" +

"Maximum tags per page:

%W\n" +

"Maximum value width:

%V\n" +

"Printing status column:

%C\n" +

"Printing -9999 on bad status:

%B\n"
