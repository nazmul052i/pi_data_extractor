Deleting a Cim-IO Logical Device Name

To delete a Cim-IO logical device name:

  1. From the Options dialog, click the logical device name which you want to delete.

  2. Click the Delete button. The Delete logical device name (the name of the selected device name) message box appears, asking if you want to delete the logical device name.

  3. Click Yes to delete the logical device name, or No to cancel the deletion. The message box closes.

    If you clicked Yes, the logical device name is deleted.

  4. Click OK. The Options dialog closes.
