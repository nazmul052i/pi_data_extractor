Viewing Entries

To view different groups of entries:

  1. Choose one of thes View buttons described below.

View All
Show all entries.

View Required
Show Required Entries only.

View Optional
Show optional Entries only.

Notes

  • One of the buttons in this set will always be selected.

  • Optionally, use menu commands to change the view. Click View | Show Entries | All, Required, or Optional.
