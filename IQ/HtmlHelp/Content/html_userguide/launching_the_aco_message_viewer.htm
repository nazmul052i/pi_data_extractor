View Messages Generated by Online Applications

You can use Aspen Production Control Web Server to view messages generated by Aspen IQ, Aspen DMCplus, and other applications as they operate online.

  • Access the Operations or Engineering view for a particular IQ.

    A table listing Messages relevant to the IQ application is displayed in the lower portion of the screen.

  • Click the Online tab and, then, from the menu bar, click Messages to display the Messages view.

    The Messages view, enables you to see messages from all online applications, or you can use filters to limit display of messages to only those that are relevant to a selected application type or area of interest.
