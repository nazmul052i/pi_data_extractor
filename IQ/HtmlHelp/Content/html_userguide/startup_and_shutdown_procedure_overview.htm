Web-Based Interface to Online Aspen IQ Applications

Use Aspen APC Manage to manage online Aspen IQ applications

Aspen APC Manage is the utility program installed on the online applications server for managing Aspen IQ and other online applications. Use it to load, start, or stop applications, set the auto-start flag, and so forth.

The following topics provide an introduction to using Aspen APC Manage, with procedures for starting or stopping applications:

  • Starting or Stopping Online Aspen IQ Applications

  • Starting or Stopping all ACO Online Products

For more details about using Aspen APC Manage, see the Aspen APC Manage Help that is included with the program.

Web-based interface for managing online Aspen IQ applications

Aspen Production Control Web Server also provides a web-based version of Aspen APC Manage.

To access the web-based interface, click the Online tab within Aspen Production Control Web Server. Then click the menu option for the Manage view. (See Note, next.) For more information, see the Aspen Production Control Web Server WebHelp.

Note:  In Aspen Production Control Web Server, access to the Manage view requires specific user role permissions.
