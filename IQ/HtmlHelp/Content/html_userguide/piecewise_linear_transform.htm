Transform Overview

This is a piecewise linear transform that performs a linear interpolation.

The transform statement uses the following form:

PWLN x1 y1 x2 y2 [x3 y3 [â€¦[xnn ynn]]]

where:

  • x1, x2, ...
    Gives a point on a line segment in the process variable domain.

  • y1, y2, ...
    Gives a point on a line segment in the transform domain.

Note:  Click the buttons to increase or decrease the number of X-Y pairs.

The input to the transform should be within the range of the line segments mapped by the x-y pairs. An input of a value outside this range will cause the transform to fail, and will mark that variable or attribute as a bad value.
