Get History File Header Information

Choose this option to display a list of history file header information. This list contains:

  • History file name

  • History file version

  • Number of tags

  • Maximum number of tags

  • Time when created

  • Time when last updated

  • Backup scheduled option

  • Backup frequency

  • Backup versions

  • Time when last backup was done

Format arguments for the fileinfo command

You can specify a different format for the list using an optional argument for the fileinfo command. The following codes are used to represent specific pieces of information:

  • %F – History file name

  • %V – History file version

  • %N – Number of tags

  • %M – Maximum number of tags

  • %C – Time when created

  • %U – Time when last updated

  • %S – Backup scheduled option

  • %P – Backup frequency

  • %D – Backup versions

  • %B – Time when last backup was done

  • \n – End of line (carriage return)

  • \t – Tab

Any other text will be reproduced as is.

Note:  The above codes are case sensitive. For example, %f will not work. Instead, use %F to obtain the history file name.

Tip:  You will typically want to end the line using the \n code (for end of line, or carriage return). Otherwise, the list will all be one line.

Default format

The default format as used by the menu option is:

"File name:

%F\n" +

"Version:

%V\n" +

"Number of Tags:

%N\n" +

"Maximum Tags:

%M\n" +

"Created:

%C\n" +

"Last Updated:

%U\n" +

"Backup Scheduled:

%S\n" +

"Backup Frequency:

%P (hours)\n" +

"Backup Versions:

%D (days)\n" +

"Last Backup:

%B\n"

Note:  The format of the timestamp values (which include %C, %U, and %B) can be modified by using the Aspen IQ History Output Options Menu - option [2] Set the date and time format.
