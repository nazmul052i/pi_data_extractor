Licensing Troubleshooting

For Aspen IQconfig and Aspen IQmodel to function properly, ensure the following when the program is loaded and running.

  • A connection to a Software License Manager (SLM) server has been successfully established.

  • The correct licenses are in place on the appropriate computers.

If these components are not found, the software will not run and a licensing error message will be displayed.
