Prediction (PR) Module

When creating a new IQ that includes a Prediction module, you must also complete the steps for creating a Prediction module.

To create a Prediction module:

  1. After clicking OK to close the Add Inferential Quality dialog, the Model Type dialog appears.

  2. Click the button next to the desired model: Algebraic Calculation Model, IQmodel Model, DMCplus Model (FIR), or Deep Learning Model, and then click OK. A dialog, corresponding to your selection, appears (either the Algebraic Calculation Model, IQmodel Model, DMCplus Model (FIR), or Deep Learning Model dialog).

  3. Complete the dialog and click OK.

    For details, refer to the following topics, depending upon the model type you selected:

    • Using an Algebraic Calculation Model for prediction.

    • Using an IQmodel Model for prediction.

    • Using a DMC3 .AIQ Model for Prediction. (Choose the IQmodel Model option, illustrated above, to begin this path.)

    • Using a DMC3 Deep Learning Inferential Model for Prediction

    • Using a DMCplus Model for prediction.
