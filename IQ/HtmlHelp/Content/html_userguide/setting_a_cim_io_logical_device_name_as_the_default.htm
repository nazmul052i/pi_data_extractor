Setting a Cim-IO Logical Device Name as the Default

To set a default Cim-IO logical device name:

  1. From the Options dialog, make sure Cim-IO is selected as the Connect protocol.

    Note: You must have added one or more logical device names before you can set this option.

  2. Click the logical device name that you want to set as the default.

  3. Click the Set as Default button. The name you selected as the default appears in the Default Logical Device field.

  4. Click OK. The Options dialog closes.
