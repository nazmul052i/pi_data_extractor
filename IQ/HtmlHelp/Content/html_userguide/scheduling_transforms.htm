Transform Overview

The transformed parameters BPRX and BAZX are calculated in the Prediction module at the same place as the current calculation of the BPR and BAZ. The PREDBIASX is calculated in the Lab and Analyzer modules where the current PREDBIAS is calculated. The Prediction module will use the PREDBIASX to calculate the BPRX and BAZX. The resultant BAZ and BPR are updated using these calculated transformed parameters.
