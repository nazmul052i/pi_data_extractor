Creating a New IQ

To add a new IQ:

  1. Click the button or open the Edit menu and select Add IQ (or, right click the Config folder and select Add IQ from the popup menu). The Inferential Quality Properties dialog box is displayed.

  2. Select the check boxes next to the types of modules you want to include in the IQ.

    Note:  If you select Prediction as a module to include, refer to Creating a Prediction module for details.

  3. To have the IQ monitored by Aspen Watch Performance Monitoring, select the Performance Monitoring check box.

    Note: The Performance Monitoring check box is enabled (for selecting or clearing) only if you first select the check box for including a Prediction module. For more information about Aspen Watch Performance Monitoring, see Enabling Performance Monitoring.

  4. Enter a name, description, and engineering units for the IQ.

  5. Click OK. The IQ is created and placed in the tree view.
