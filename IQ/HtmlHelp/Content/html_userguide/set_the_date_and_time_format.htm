Set the Date and Time Format

Choose this option to modify the format of timestamp values printed by the IQextract program. To change the format of the date and time, you must have a copy of the standard ANSI C library reference that has a full list of formatting conversion specifiers for the strftime C function.

Following are some examples for setting the date and time format:

-- Aspen IQ History Output Options Menu 1.2.00 --

1> List history output options

2> Set the date and time format

3> Set the maximum number of tags per page

4> Set the maximum value width

5> Set the status column printing option

6> Set the printing of -9999 for bad status option

7> Aspen IQ History Output Menu...

Enter a choice (1-7) [7]: 2

 

Enter output time format [%m-%d-%Y %H:%M:%S]:

Time values will be formatted as: 06-17-1999 16:05:16

 

Enter again (Y/N) [N]: y

 

Enter output time format [%m-%d-%Y %H:%M:%S]: %A %B %Y %I:%M:%S %p

Time values will be formatted as: Wednesday June 1999 04:06:43 PM

 

Enter again (Y/N) [N]: y

 

Enter output time format [%m-%d-%Y %H:%M:%S]: %d-%b-%Y %H:%M:%S

Time values will be formatted as: 17-Jun-1999 16:07:27
