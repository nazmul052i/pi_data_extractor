Web-Based Interface to Online Aspen IQ Applications

See Also

Common Tasks for Online Applications

Aspen Production Control Web Server provides a web-based interface for monitoring and tuning online Aspen IQ applications.

To access the web-based interface to online Aspen IQ applications:

  1. In Aspen Production Control Web Server, click the Online tab.

  2. In the NavBar, see the items listed under IQ Applications.

    If necessary, expand the tree view in the NavBar to see all IQs that are included in each online Aspen IQ application.

  3. From the menu bar, click Overview, Operations, or Engineering, as desired to see these views.

    Note:  Access to the Operations and Engineering views in Aspen Production Control Web Server requires that you have appropriate user permissions.

  4. For more information, see the Aspen Production Control Web Server Help, along with the ToolTips and Aspen IQ Entry Dictionary Help topics, which are made available from links in the web-based views.

To set up the web-based interface to online Aspen IQ applications:

  1. See instructions in the Aspen Manufacturing Suite Advanced Process Control Installation Guide and the APC Configuration Guide that apply to setting up Aspen Production Control Web Server and online products (including Aspen Inferential Qualities, or Aspen IQ Online).

  2. To learn how to add Aspen IQ applications to Aspen Production Control Web Server, see the topics "Adding Online Applications to the Web Server" and "Site Wide Settings" in the Aspen Production Control Web Server WebHelp.

When connections are established between the the Aspen IQ Online server and Aspen Production Control Web Server, then the IQ Applications list (in the NavBar of Aspen Production Control Web Server) automatically includes all online Aspen IQ applications after they are deployed.
