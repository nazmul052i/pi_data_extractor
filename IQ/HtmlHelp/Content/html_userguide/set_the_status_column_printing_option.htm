Set the Status Column Printing Option

Choose this option to enable or disable the printing of status column in the CSV output file. When this option is enabled a status of "Good" or "Bad" is printed next to every data value, representing the quality of the value, as stored in the history file.

Examples are provided for the four possible cases:

Option 1: Status column on, -9999 on bad status on.

Printing status column: on

Printing -9999 on bad status: on

Note: This is off by default since Printing status column is on.

-9999 Good 3/11/99 11:00

-9999 Good 3/11/99 11:06

150.403 Good 3/11/99 11:36

NaN Bad 3/11/99 11:54

NaN Bad 3/11/99 12:00

NaN Bad 3/11/99 12:06

151 Bad 3/11/99 12:12
