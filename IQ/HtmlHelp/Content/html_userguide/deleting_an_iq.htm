Deleting an IQ

To delete an IQ:

  1. In the tree view, click the IQ you want to delete.

  2. Then, click the button, or open the Edit menu and select Delete IQ (or, right-click the desired IQ folder and select Delete IQ from the popup menu).

    A message appears, asking if you want to delete the IQ.

  3. Click Yes to remove the IQ.
