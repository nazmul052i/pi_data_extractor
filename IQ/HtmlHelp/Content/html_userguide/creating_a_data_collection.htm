Creating a Data Collection Module

See Also

Data Collection Module

To create a data collection module:

  1. In the tree view, click the desired DC (Data Collection) folder.

  2. Click the button or open the Edit menu and select Properties. The Data Collection Properties dialog appears.

  3. To add data points, click the Add Point button. Go to step 4.

    To add points using an ASCII text file, click the Load From File button. The Select a File dialog appears. Change to the desired path (drive/directory) and select the desired text file (*.txt).

  4. If you selected the Add Point button, complete the following fields: Tagname, Label, Source, Description, Device, Eng Units, and Unit. The Source and Device fields have pull down menus for entry selection options. Click OK to add the point. The point is added to the list.

  5. To remove a point, click the point in the list and then click the Delete button. A message appears, asking if you want to delete the point. Click Yes.

  6. Click OK when done with the Data Collection Properties dialog.
