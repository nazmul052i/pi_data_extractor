Enabling Performance Monitoring

The Inferential Quality Properties dialog box in IQconfig includes the following check box:

  • Enable Performance Monitoring in Aspen Watch
Note: Select the check box for the above option only if Aspen Watch Performance Monitor and Aspen Production Control Web Server are installed and operating at your site.

Enable Performance Monitoring in Aspen Watch

If this check box is selected for an IQ that you configure in IQconfig, then the IQ is made available for continual performance monitoring through analyses of historical data and comparisons against benchmark values. (Also see AWENB in the Aspen IQ Entry Dictionary.)

The web-based user interface for this feature is provided by Aspen Watch Performance Monitor (Aspen Watch) and the key performance indicator (KPI) plots that are displayed in the History tab of Aspen Production Control Web Server. Each cell in a KPI plot provides a color-coded indicator of the performance of a particular facet of the IQ's characteristics.

Important: See Unique Application Names Requirement for Aspen Watch Performance Monitor.
