Adding/Modifying Cim-IO Logical Device Names

To add a Cim-IO logical device name:

  1. From the Options dialog, make sure Cim-IO is selected as the Connect protocol.

  2. Click the New button. The New Cim-IO Logical Device dialog box appears.

  3. Type a name in the Enter a Cim-IO logical device name field.

  4. Click a button next to the desired Tag Name Case Sensitivity option, either Force to Upper Case, Force to Lower Case, or No Change to Case.

  5. Click a button next to the desired Tag Name Space Sensitivity option, either Compress All Extra Spaces, Trim Leading/Trailing Spaces, or No Change to Spacing.

  6. Click OK when done. The Options dialog appears again, with the new logical device name added to the list.

To modify a Cim-IO logical device name:

  1. Double-click the logical device name you want to modify (or single-click it then click the right mouse button to display a pop up menu and select Modify). The New Cim-IO Logical Device dialog appears.

  2. Follow steps 3-6 listed above.
