Using Help

Here are some tips about using Help in this AspenTech application.

Opening Help

The following options are available for opening online Help:

  • From any menu bar in which the Help command is available, click the Help command for Help which opens in a new Browser window.

  • For Application Help, click the Application Help option on the Footer Bar or click the help buttonto access the Help system’s Table of Contents.

  • If a Help link exists within an application in the Browser’s pane, click that Help link or press F1 for Page-level Context Sensitive Help.

Adjusting the Window

Drag the outside edge of the Browser to adjust its size. Inside the Help window, drag the vertical pane divider to adjust the size of the two panes. You can click the hide button to hide the contents pane entirely.

If a topic is too large to fit in the window, you can use the horizontal and vertical scroll bars to view the rest of the topic, or resize the window. If you resize the help window, the new size is retained the next time you open help.

Finding a Help Topic

At the top of the Table of Contents pane, click one of the following tabs to find the desired Help topic:

The Contents tab displays a list of the topics included in Help. Click an expand icon to open a list of topics within that book. Click a topic icon to open a topic.

The Index tab displays an alphabetical list of the keywords attached to the topics. Type a keyword to search the list. Select a keyword to display the topic.

You can also use the Search feature to search the full text of the help. Type your query in the search box at the upper right of the help window and press Enter or click the search button:

When typing a search query:

  • If you type multiple separate words, the query will show topics containing all of those words.
  • If you type a phrase in quotes, the query will show topics containing those words adjacent to one another.
  • If you type words with the word or between them, the query will show topics containing either of these words.
The Help Toolbar

Use the buttons immediately above the topic to:

  • Print the help topic.
  • Expand or collapse all expanding text within the help topic
  • Remove or show highlighting of search terms, if you opened the current topic from a search.

Use the browser buttons at the top of the window to:

  • View the help home page.
  • Navigate back to the previously viewed topic.
  • Navigate forward (after having navigated back).
Help Graphics

If you pass the cursor over a graphic in the help file, and the cursor changes to a pointing hand icon , click the area to display information related to the graphic.
