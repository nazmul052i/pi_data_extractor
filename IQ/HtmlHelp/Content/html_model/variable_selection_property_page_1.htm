Variable Selection Property Page

The Variable Selection property page appears in the Expert Parameters dialog for the Select Variables step:

The Cascaded Variable Selection section on the Variable Selection tab allows you to specify parameters for cascaded variable selection.

  • The Minimum Selection Rate: edit box indicates the minimum fraction of inclusion in the top portion of the population that a variable must have in order to continue to be considered by the selection algorithm.

  • The Increase in Selection Rate: edit box specifies an amount that the Minimum Selection Rate should be increased at each step.

  • The Portion of Population to Test: edit box indicates the fraction of the most fit members of the population to test at the end of each step.

  • The Maximum Cascade Steps: edit box indicates the maximum number of steps for Cascade Variable Selection. At each step, the GA is called, using convergence criteria specified on the GA [1] tab.
