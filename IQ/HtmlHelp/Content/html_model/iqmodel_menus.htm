IQmodel Menus

In addition the action step boxes in the Commands window, IQmodel capabilities can be accessed through menus in the IQmodel menu bar.

The current IQmodel context (for example, whether a model file is loaded, or whether a graph or grid is visible) determines which IQmodel menus, as well as what commands in each menu, are available.

Note:  Standard Windows menus are discussed in the section Using Common Windows Functions.
