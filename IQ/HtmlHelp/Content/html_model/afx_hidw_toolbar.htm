Toolbar

The toolbar is displayed across the top of the application window, below the menu bar:

To hide or display the Toolbar, choose Toolbar from the View menu (ALT, V, T).

Toolbar buttons provide quick mouse access to many tools used in IQmodel:

Open a new document.

Open an existing document. IQmodel displays the Open dialog box, in which you can locate and open the desired file.

Save the active document or template with its current name. If you have not named the document, IQmodel displays the Save As dialog box.

Print the active document.
