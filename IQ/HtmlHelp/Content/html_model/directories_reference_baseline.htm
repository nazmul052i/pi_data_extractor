Root and program directories for typical installations of Aspen Inferential Qualities (Aspen IQ)

Various Help topics use the following terms to symbolize the paths of root and program data directories for typical installations:

ONLINEAPPS

Directory on the applications server for hosting deployed online applications. Typically, this is...

C:\ProgramData\AspenTech\APC\Online

PROGRAMDATA

Directory for example files, templates, project configuration files, and so forth—as used by the Aspen IQ programs. Typically, this is...

C:\ProgramData\AspenTech\APC\<version>\Builder

PUBLICDOCUMENTS_PROGRAMDATA

Directory for example files and tutorial documents. On machines running newer versions of Windows™ operating system, this is...

C:\Users\Public\Documents\AspenTech\APC\<version>\Builder
