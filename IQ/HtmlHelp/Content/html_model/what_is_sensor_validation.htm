What is Sensor Validation?

Sensor validation, like inferential sensing, involves creating process models from plant data. A sensor validation application continuously monitors plant data for poor quality or missing values. When anomalous values are detected, they are replaced with estimated values generated by a mathematical model that reflects the relationships among data from the (now faulty) sensor and other available plant DCS (Distributed Control System) values acquired under normal operating conditions. In contrast to inferential sensor models, redundant variables (that is, highly correlated variables) are essential for good sensor validation models.
