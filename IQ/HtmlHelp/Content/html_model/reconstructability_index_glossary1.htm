Reconstructability Index

The Reconstructability Index is an indication of how accurately a sensor value can be reconstructed by the sensor validation model when a sensor fault is detected. A reconstructability index of 1 indicates that the faulty sensor value can be perfectly reconstructed (given the current dataset). A reconstructability index of 0 means that the reconstruction error based on the modelâ€™s reconstructed value is no better than the reconstruction error that would result if the average value for the sensor (over the training dataset) was used as the reconstructed value.
