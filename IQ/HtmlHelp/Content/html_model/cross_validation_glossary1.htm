Cross Validation

Cross Validation is a technique used when determining how many latent variables to include in a PLS model. When Cross Validation is selected, the training dataset is partitioned into a user-specified number of subsets. Then each subset is consecutively marked for validation (i.e., not used in model training) and a PLS model is built with the remaining data. The variance that the resulting PLS model captures when it is run with the validation subset is recorded as a function of the number of latent variables in the PLS model. The optimum number of latent variables finally selected for the PLS model is the number that maximizes the average of the validation variances.
