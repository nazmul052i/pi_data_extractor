Uniform Crossover

Uniform Crossover is a genetic algorithm reproduction technique that creates two children from two parents, with a probability of 0.5 that corresponding parental chromosomes (bits) are swapped in each child. To implement Uniform Crossover, a template whose length is equal to the number of chromosomes (bits) in a parent is randomly generated. For each bit position in the parents, the template indicates which parent will contribute its value in that position to the first child; the second child then gets the corresponding bit from the other parent.
