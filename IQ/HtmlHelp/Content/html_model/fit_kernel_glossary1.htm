Fit Kernel

The Fuzzy PLS algorithm models the relationship between the PLS model's input and output scores through an arbitrary nonlinear function. This nonlinear function is constructed as a sum of Gaussian type fit kernels spaced at equal distances or according to the distribution of the PLS input scores. This function is displayed with PLS input (x) and output (y) scores in the Model Analysis Graph.
