Close command (File menu)

Use the Close command to close all windows related to the active document (model). IQmodel suggests that you save changes to the document before you close it. If you close a document without saving, you lose all changes made since the last time you saved it. Before closing an untitled document, IQmodel displays the Save As dialog box and suggests that you name and save the document.

Note:  You can also close a model by using the Close button on the document's window.
