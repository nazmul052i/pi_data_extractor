Aspen IQmodel Licensing Troubleshooting

If Aspen IQmodel fails due to a licensing problem, it is necessary to correct the licensing problem before the application can be restarted. Depending on the nature of the licensing problem there are several different actions to take.

Select the type of error message that is displayed, from the list below, and then follow the instructions provided:

Note:  ALM server administrators should consult the troubleshooting section of the Aspen License Manager Installation and Reference Guide for more information.
  • System error
  • Client invalid
  • Unable to read file
  • License authorization failure
  • Code or license key corrupt
  • License start date not yet reached
  • Invalid client or server system date
  • License request denied
  • License has expired
  • Invalid feature
  • No license file found for feature
  • No licenses are currently available
  • License server is down or not responding
  • License server initializing; please wait
  • Network error: unable to send message to license server
