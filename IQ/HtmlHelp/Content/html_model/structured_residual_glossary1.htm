Structured Residual

A Structured Residual is the value that results from the dot product of the Structured Residual Transform vector for a tag and the Equation Error vector for the sensor validation model. Under normal operating conditions, Structured Residual values for tags in a sensor validation model are small and have zero mean. When a sensor fault occurs, the Structured Residual value increases for all sensors except the one that caused the fault; thus, the faulty sensor can be readily identified.
