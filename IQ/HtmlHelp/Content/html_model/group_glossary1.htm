Group

A Group is a collection of independent variables whose dead time and/or dynamic filter relationships with the modelâ€™s dependent variable have essentially the same value. A group is represented by one variable in GA dead time and dynamics searches (i.e., all variables in a group have the same dead time and/or filter value). In addition, for Cascaded Variable Selection input variables can be grouped together and the user can specify how many variables in each group should be selected for the model.
