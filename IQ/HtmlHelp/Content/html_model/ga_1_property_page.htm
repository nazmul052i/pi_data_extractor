GA[1] Property Page

The GA [1] property page is visible in the Expert Parameters dialog for the Align Data, Select Variables, and Analyze Dynamics steps. The GA [1] page is one of two property pages that provide access to parameters that govern execution of the Genetic Algorithm:

The General section on the GA [1] tab allows you to specify general parameters for the genetic algorithm.

  • The Execution Count: edit box indicates the number of times the GA will be executed. Results that reflect the best population out of all the executions are displayed in the main dialog from which the Expert Parameters dialog was activated; however, results for all executions can be viewed in the grid that is accessible after the associated main dialog is closed.

  • The Random Seed: edit box allows you to specify a seed value for the random number generator used by the GA. In general, changing this value will result in different GA run patterns and thus will yield different results.

The Population Size section allows you to specify population size multipliers for the genetic algorithm.

  • The Population Size Factor: edit box allows you to specify a multiplier that is applied to the basic population (number of unknowns) to yield the actual population size. In other words, the population size actually used in the GA is the number of unknowns times the population size factor.

  • The Overinitialization Factor: edit box allows you to specify an additional multiplier factor used to increase the size of the first generation only (this factor cannot be less than 1.0).

The Convergence section allows you to specify convergence parameters for the genetic algorithm.

  • The Maximum Iterations: edit box specifies the maximum number of iterations the GA will execute (i.e., the maximum number of generations that will be produced).

  • The Tolerance: edit box allows you to specify the tolerance value for GA convergence. The difference in average fitness between succeeding generations must be smaller than this value to terminate the GA.

  • The Patience: edit box allows you to specify a patience parameter (the number of additional consecutive times the tolerance criterion must be met, after the first, before the GA terminates).
