GA[2] Property Page

The GA [2] property page is visible in the Expert Parameters dialog for the Align Data, Select Variables, and Analyze Dynamics steps. The GA [2] page is one of two property pages that provide access to parameters that govern execution of the Genetic Algorithm:

The Fitness Evaluation section on the GA [2] tab allows you to specify the fitness parameter.

  • The Favor Best/Average: edit box allows you to specify the ratio that represents the fitness of the best member of the population compared to the average fitness of the population. Fitness values in the initial population are scaled such that this ratio is satisfied.

The Parent Selection section allows you to specify parent selection techniques for the genetic algorithm (refer to the Glossary for explanations of these techniques).

  • If the Roulette Wheel radio button is selected, the Roulette Wheel technique is used to determine parents for the next generation.

  • If the Modified Roulette Wheel radio button is selected, a Modified Roulette Wheel technique is used to determine parents for the next generation.

  • If the Gaussian Distribution radio button is selected, a Gaussian distribution technique is used to determine parents for the next generation.

  • The Elitism Copies: edit box allows you to specify an elitism value (i.e., how many of the most fit members of a population should be directly copied into the next generation).

The Reproduction section allows you to specify a reproduction technique and other parameters that govern how new populations are generated after parents are selected.

  • If the Uniform Crossover radio button is selected, Uniform Crossover is used to produce child populations. Uniform Crossover creates two children from two parents, with a probability of 0.5 that corresponding parental chromosomes (bits) are swapped in each child.

  • If the One-Point Crossover radio button is selected, One-point Crossover is used to produce child populations. One-point Crossover creates two children from two parents by swapping all of each parent’s chromosomes (bits) after some randomly selected point.

  • If the Two-Point Crossover radio button is selected, Two-point Crossover is used to produce child populations. Two-point Crossover creates two children from two parents by swapping each parent’s chromosomes (bits) between two randomly selected points.

  • The Crossover Rate: edit box allows you to specify the Crossover Rate. The Crossover Rate indicates how frequently crossover occurs in a population.

  • The Mutation Rate: edit box allows you to specify the Mutation Rate. The Mutation Rate indicates how frequently mutation occurs.

  • If the Tournament with Parents check box is checked, in the final selection of population members for the next generation, children compete with their parents for inclusion in the new generation.
