Size command (System Menu)

Use the Size command to display a four-headed arrow so you can size the active window with the arrow keys:

After the pointer changes to the four-headed arrow:

  1. Press one of the Direction keys (left, right, up, or down arrow key) to move the pointer to the border you want to move.

  2. Press a Direction key to move the border.

  3. Press Enter when the window is the size you want.

As an alternative to the above steps, you can resize the window using the mouse to drag the size bars at the corners or edges of the window.

Note:  The Size command is not available if you maximize the window.
