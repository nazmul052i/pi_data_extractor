Overview of Expert Parameters Dialogs

Each main model step action dialog has an associated Expert Parameters dialog that allows you to set additional parameter values for operations relevant to the step.

Note:  In both Build Mode and Run Mode, the Expert Parameters dialog for the Specify Data step is accessed through the Expert button on the File Format dialog, which pops up when a new file is loaded, or when the Format... button on the Data Specification dialog is clicked.

In each Expert Parameters dialog, parameters are grouped on one or several tabbed property pages. When the Expert Parameters dialog is opened for a particular step, only property pages appropriate for that step are visible.
