About This Guide

This IQmodel Help System provides the information needed to develop and deploy empirical Inferential Sensor and/or Sensor Validation models that are built using plant data.

Who Should Read This Guide

This Guide is intended for anyone who creates or uses inferential sensor or sensor validation models. This includes system integrators and process control engineers as well as programmers.

Prerequisites

In addition to a basic understanding of the processes you are modeling, this Guide assumes that you are familiar with:

  • The standard Microsoft® Windows™ user interface.

  • The characteristics and capabilities of your particular development and runtime environments.

  • The format of files generated by plant process data collection and logging facilities.

Additional Information

Before you start using Aspen IQmodel software, please refer to the Aspen Manufacturing Suite Advanced Process Control Release Notes for any additions or corrections to the information in this Guide.
