Title Bar

The title bar is located along the top of a window:

To move the window, drag the title bar.

Note:  You can also move dialog boxes by dragging their title bars.

A title bar may contain the following elements:

  • Name of the application (IQmodel)

  • Name of the model (untitled_X until the model is saved)

  • Name of the current view (for example, Commands)

A title bar may also contain the following buttons:

  • Application Control – menu button

  • Document Control – menu button

  • Maximize

  • Minimize

  • Restore

  • Close
