What is Inferential Sensing?

Inferential sensing is the science of using mathematical models to generate appropriate values for difficult- or costly-to-measure process properties.

In an inferential sensing application, the value of a critical process variable or product property is inferred from the values of other continuously measured process variables and their relationships. This inference yields a model whose corresponding stream of data values simulates the expected behavior of a sensor monitoring the critical variable or property.

Generally, it is desirable to eliminate redundant variables in an inferential sensor model.
