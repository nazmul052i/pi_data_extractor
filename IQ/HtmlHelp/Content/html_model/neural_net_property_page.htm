Neural Net Property Page

The Neural Net property page shown below appears in the Expert Parameters dialog for the Build Inferential Sensor step. Neural Net parameters govern the construction and execution of the neural network that is used when the Hybrid: PLS - Neural Network model is selected:

The Architecture section allows you to specify general parameters for the neural network.

  • The Max Hidden Units: edit box indicates the maximum number of hidden units that the neural network can contain. If the training dataset is noisy, overfitting of the neural network to the training data can be avoided by limiting the number of hidden units.

  • If the Direct Connections check box is checked, the input layer of the neural network will be directly connected to the output layer.

  • If the Cascaded Connections check box is checked, previously established hidden units will be connected to new hidden units as they are added to the neural network during construction.

The Training section allows you to specify parameters that govern neural network training.

  • The Noise Level: drop-down list allows you to specify how noisy the input dataset is.

    Select the description that most closely matches the noise characteristics of the input dataset.

  • The Learning Rule: drop-down list allows you to specify the learning rule for the neural network.

    • The Kalman Filter learning rule considers neural network weights as states and the desired outputs as observations within a discrete state-space transition framework. Standard nonlinear Kalman filter theory is used to obtain the best estimate of the weights based on the available training data. Kalman filter training is most typically used with noisy data.

    • The Adaptive Gradient learning rule is a form of back-propagation that uses a variable metric to adjust weights during training. The error signal is the gradient of mean square error with respect to the adjustment. The Adaptive Gradient rule is appropriate when input data is not particularly noisy.

The Incremental Hidden Units section allows you to specify parameters that govern the incremental addition of hidden units to the neural network.

  • The Minimum: edit box indicates the minimum number of hidden units that can be incrementally added at a time as the neural network is constructed.

  • The Maximum: edit box indicates the maximum number of hidden units that can be incrementally added at a time as the neural network is constructed.
