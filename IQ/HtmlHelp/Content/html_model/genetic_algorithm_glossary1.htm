Genetic Algorithm

A Genetic Algorithm is a multi-variate optimization algorithm that finds optimal solutions for complex problems using relatively simple encoding and reproduction mechanisms that mimic biological evolution in nature. The basic steps in a genetic algorithm are:

  1. Randomly initialize a population (assign fitness values to members).

  2. Evaluate the fitness of each member of the population using a cost function.

  3. Create new members for the next population by selecting parents, mating parents, and applying mutation.

  4. Optionally delete existing population members and insert new members into the population, based on fitness rankings.

  5. Determine whether to stop, based on termination criteria; if it is not time to stop, return to step 2.
