Welcome to Aspen IQmodel

Thank you for selecting Aspen IQmodel software. You now have a powerful empirical modeling tool which you can use to build linear or non-linear models of process variables that are infrequently monitored or difficult and costly to measure directly.

In addition, IQmodel is ideally suited for developing robust sensor validation models that can not only detect, identify and classify hardware sensor failures, but can also generate replacement values so that critical process manufacturing operations can continue while hardware maintenance is performed.
