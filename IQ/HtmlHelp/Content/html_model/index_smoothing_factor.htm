Index Smoothing Factor

The Index Smoothing Factor is the coefficient of a first order moving average filter, which is used to smooth the Filtered Fault Detection Index (FFDI), and the Normalized Squared Filtered Structured Residuals (NSFSR). The FFDI and the NSFSR are part of the Sensor Validation Model.

The Index Smoothing Factor must be greater than or equal to zero, and less than 1.0. A value of 0.0 means no smoothing. A value very close to 1.0 means substantial smoothing. An initial value for the Index Smoothing Factor is established when a model is built; however, the Index Smoothing Factor can be changed when a model is deployed online if operating conditions warrant.

As with any smoothing factor, a balance has to be made between smoothing out the noise while still maintaining the information in the indices.
