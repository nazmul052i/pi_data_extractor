The Variable Selection Dialog

The Variable Selection dialog reflects the process of identifying an optimal number of input variables whose values can be combined to represent the target process variable. Variables can be explicitly selected by the user, or one of IQmodel’s automated search mechanisms can be used.

Note:  The Variable Selection dialog (i.e., the Select Variables box) is only available if the corresponding check box was checked (in the Model Defaults dialog) at the beginning of the model building process.

Note:  Results from the Select Variables step are also available in the Variable Selection Results grid.

Variables Section

The Variables section contains a summary of information about model independent variables.

  • The Name column contains independent variable tag names.

  • The Group column contains the variable’s Group name (all variables are in the Default group unless you explicitly define another Group and assign variables to it).

  • The RSQ column contains the value of the R Squared statistic for the corresponding variable.

  • The Status column indicates processing status for the variable. GA means the genetic algorithm determines whether or not to include the variable. Active means the User has explicitly included the variable. Off means the User has explicitly excluded the variable.

  • The Min column contains the current minimum number of variables in the corresponding Group that must be Active.

  • The Init column contains the current maximum number of variables in the corresponding group that can be Active in the initial population for the genetic algorithm.

  • The Glob column contains the current maximum number of variables in the corresponding group that can be Active in any population other than the initial population. The value of Glob must be greater than minimum number of variables required from the group, and it cannot exceed the number of variables in the group.

  • The Result column contains the final status (Active or Off) of the corresponding variable (after the genetic algorithm search has been performed, or as a result of the user specifying the status).

On the Statistics: line:

  • The edit box labeled Variables contains the total number of independent variables;

  • The edit box labeled "Free" Variables contains the number of free, or effective variables (i.e., the number of variables that the GA will use during variable selection);

  • The edit box labeled Groups contains the number of groups currently defined.

The Model Status: edit box indicates the current model status at this step (Invalid, Ready, Out of Date).

  • Invalid means that the operations associated with this step have not yet been performed.

  • Ready means that the operations associated with this step have been completed and results of this step are ready for use in subsequent model steps.

  • Out of Date means that operations associated with this step have been performed and results are available for use in subsequent steps; however, one or more parameters associated with this step have been changed since the operations were performed.

The Cascade section of the Variable Selection dialog indicates the current minimum selection rate (the minimum fraction of inclusion in the top portion of the population that a variable must have in order to continue to be considered by the selection algorithm).

The Variable Selection Properties Dialog

The Variable Selection Properties: dialog is accessed by highlighting a name or names in the Variables section of the main Variable Selection dialog, and then right-clicking and selecting Properties... from the menu, as illustrated below:

The Variable Selection Properties: dialog allows you to change parameters that govern how variable selection is performed.

Note: The names of all variables in the Group corresponding to the variable that was highlighted when Properties... was selected from the pop-up menu appear in the Variables list view.

  • When the Force On radio button is selected, the status of all variables listed in the Variables list view will be set to Active, and the variables will be included in the model.

  • When the Force Off radio button is selected, the status of all variables listed in the Variables list view will be set to Off, and the variables will be not be included in the model.

  • When the GA Search radio button is selected, a genetic algorithm will be used to select the variables to include in the model. Also, the Min:, Max (Init):, and Max (Glob): edit boxes will be enabled to allow you to enter values for these parameters. The current values for these variables are displayed in the main Variable Selection dialog as described earlier).

    • The Min: edit box is used to specify the minimum number of variables in the corresponding Group that must be Active. The value must be greater than 0.

    • The Max (Init): edit box is used to specify the maximum number of variables in the corresponding group that can be Active in the initial population for the genetic algorithm.

    • The Max (Glob): edit box is used to specify the maximum number of variables in the corresponding group that can be Active in any population other than the initial population.

Creating and Deleting Groups

If several input variables are known to have similar influences on the dependent variable, they can be combined into a Group for the purpose of selecting model variables. All variables in a group are treated as one variable during the genetic algorithm search for model latent variables.

To create a group:

  1. Highlight (in the Variables section of the Variable Selection dialog) the names of variables you want to include in the group, as illustrated below:

    Note:  If the names are not contiguous, you can control-click to select multiple non-contiguous variables.

  2. Right-click on a highlighted name and select Group... from the pop-up menu.

    The Group: dialog, shown below, allows you assign a name to the new group:

    • The list view in the Variables section lists the names highlighted when you clicked Group... on the pop-up menu.

    • The Name: list box in the Group section is used to enter a name for the new group. The drop-down list contains the names of all currently defined groups.

  3. Enter the name for the new group in the edit portion of the Name: list box.

To delete a group when the main Variable Selection dialog is visible:

Note:  When a group is deleted, the members of the group are placed back in the Default group. The Default group cannot be deleted.

  1. Highlight any variable name that is in the group you want to delete.

  2. Right-click to activate the pop-up menu.

  3. Select Group... to access the Group: dialog.

  4. Locate the name of the group you want to delete in the Name: list box.

  5. Delete the name, using the Delete or Backspace keys to erase the name from the edit portion of the Name: list box.
