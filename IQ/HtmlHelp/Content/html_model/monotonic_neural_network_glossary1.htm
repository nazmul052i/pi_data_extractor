Monotonic Neural Network

Monotonic neural networks provide inferential models with guaranteed monotonicity. This means that the direction of response of the inferred variable to any change in an independent model variable is guaranteed to be consistent at all operating points. An example of where this is important is in using an inferential model for gain scheduling of a controller; if a monotonic neural network is used, it is guaranteed that the controller gain will not change sign at any operating point.

In mathematical terms, given a multiple input, single output inferential model y= f(x1, x2, .., xn), the relationship between y and a given input variable xi is said to be "positive monotonic" if ∂y / ∂xi > 0 for any fixed value of the input variables. That is, no matter what operating point the system is at, increasing the independent variable xi is guaranteed to lead to an increase in the output y.
