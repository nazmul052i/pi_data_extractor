Normalized Squared Filtered Structured Residual

A Normalized Squared Filtered Structured Residual (NSFSR) is an index associated with each sensor (input variable) in a model. Thus a sensor validation model has a set of these NSFSRs, all of which are designed to be insensitive to a fault in the corresponding sensor. When a fault occurs in a particular sensor, its NSFSR should remain below the normal NSFSR threshold, while the NSFSRs for all other sensors should rise above the threshold. The set of NSFSRs is calculated each model cycle; if any NSFSRs exceed the threshold, additional indices are computed to help identify the type of sensor fault that occurred.
