Search for Help On Command (Window Menu)

Use the Search for Help On command to display the Help Topics: dialog. The visible page will be the last page you accessed in the dialog. If this command is used to access the Help Topics: dialog the first time, the Index page for the IQmodel Userâ€™s Guide will be visible.

If the Index page is not visible, click on the Index tab to make the Index page visible.

To search for Help on a specific term:

  1. In the Index tab, type the term you wish to search for.

    As you type, partial matching choices are displayed in list.

    If a matching choice is found, double-click the term or phrase.

  2. Alternatively, click the Search tab, and type the single word that you wish to search for.

    As you type, a list of Help topics that include the search word is displayed.

    Click each topic title in the list to display the topic.

    Look for the search term highlighted within the Help topic.
