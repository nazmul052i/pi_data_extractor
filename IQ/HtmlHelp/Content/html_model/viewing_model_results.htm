Viewing Model Results

To the right of each box in the Commands window that represents a step in the model building process, a drop-down menu button provides access to grid and (for most steps) graph views of the results of the step.

If a model is loaded in IQmodel, grid view -specific and graph view -specific commands are added to the View menu in the IQmodel menu bar.

Also, if the loaded model incorporates sensor validation, an Expert command in the drop-down menu to the right of the Build / Run Sensor Validation box provides access to additional grid and graph views that provide more detailed sensor validation model information.

Note:  The commands available in the menu that follows the Build / Run Sensor Validation box depend on whether IQmodel is in Build Mode or Run Mode.
