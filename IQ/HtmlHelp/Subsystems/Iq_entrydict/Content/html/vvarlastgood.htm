VVARLASTGOOD

The last validated value with a validation status VVARVLDSTS of 1 (good) is stored in VVARLASTGOOD. This value is used as the basis for checking for rate of change and frozen signal violations.

Applies to:  

Validation variables

Data Type:  

Float

Access:  

Write Only

Range:  

No Restrictions

Default:  

0.0 (Required Entry)

Keywords:  

LOCAL WRITE
