ABIASNEWPCT

New analyzer bias percentage calculated during last successful analyzer bias update. This entry is used when relative analyzer bias updating is enabled (AZUPDREL = 1).

Ordinarily IQ calculates a bias in engineering units and applies the same bias to the inferential over the prediction's entire range. With the relative analyzer bias option enabled, IQ will calculate the percentage bias and then apply a constant percentage over the prediction's range. This option may be useful with processes that have transitions over a wide range of prediction values. This option will not work well with prediction values that are near zero because the relative bias will be disproportionately large.

Once a relative bias is calculated the absolute value of the bias is stored in PABIASNEW. Although ABIASNEWPCT is not historized it is easily calculated from the previous lab data as 100*PABIASNEW/PAZUAVGUAZ. With the relative analyzer bias option enabled, ABIASNEWPCT will remain constant between analyzer updates and ABIASNEW will change each execution of the prediction module as the unbiased prediction changes.
