VVARRAWSTS

Raw status value for validation. The source of this value is set by IQ and depends on the variable that is being validated. For input validation (for example, CVARPNTVAL), the status would come from CVARPNTSTS which, in turn, can be obtained from a DCS point. For output validation (for example, BPR), the status would come from BPRSTS, which is based on the status of the inputs to the model calculation and the status of the prediction bias. If the raw status is bad, then the final validation status VVAROUTSTS is set to bad.
