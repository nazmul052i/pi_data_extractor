LabStatus

The combined lab status is determined from three possible sources as the lab sample proceeds through the configured Lab Data Collection (LDC) and Lab Update (LBU) modules. The table below indicates the value source (the entry name and value) that corresponds to the combined status. The value range in this table also indicates where the status originated from. For example, a sample must be collected successfully (indicated by PLDCSTSNUM) in the LDC module before it can be used in the LBU module to do validation (indicated by PLBUVLDNUM) and bias update (indicated by PLBIASNUM). An application may or may not have lab validation or bias update configured, so those checks might not occur.

Value Text Description Value source
0 No Sample No Sample has been entered by the operator Lab Data Collection Statuses below
PLDCSTSNUM (0)
1 Waiting The lab sample time was stored but the value has not been entered. There can be multiple lab samples waiting to be processed. PLDCSTSNUM (1)
2 Stored Lab sample is read from the NEWLAB and the New Lab Value Available Flag (AVLFLAG) is set to 1. The Lab Data Collection (LDC) module responds by changing the status of the corresponding lab sample from "Waiting" to "Stored" and resets the AVLFLAG back to zero (0). PLDCSTSNUM (2)
3 Canceled Lab sample was cancelled using the Lab Sample Cancel Flag (CNCLFLAG). PLDCSTSNUM (3)
4 In Lab This condition occurs when the Bias Update module is stopped or running at a much lower frequency. This is not a commonly seen status. PLDCSTSNUM (4)
9 Not Checked This means that validation is not configured for this IQ. Lab Validation Statuses below
PLBUVLDNUM (-1)
10 Good Lab Sample has passed all validation checks and is ready for use in the bias update module (if configured). PLBUVLDNUM (0)
11 Load Failed The set of available raw lab values around the sample time were not successfully read from the lab data collection history file. PLBUVLDNUM (1)
12 No Sample This is a default state for the validation module when it is waiting for a lab sample. PLBUVLDNUM (2)
13 Above Maximum Limit Lab Sample exceeds the Lab Maximum Limit (LVMAXLIM). PLBUVLDNUM (3)
14 Below Minimum Limit Lab Sample exceeds the Lab Minimum Limit (LVMINLIM). PLBUVLDNUM (4)
15 Bad: Inc. ROC Limit Lab sample change is greater than the Lab increasing rate of change limit (LVROCLIMI) and LVROCLIMI is greater than zero. PLBUVLDNUM (5)
16 Bad: Dec. ROC Limit Lab sample change is greater than the decreasing rate of change limit (LVROCLIMD) and LVROCLIMD is greater than zero. PLBUVLDNUM (6)
17 Bad: Unsteady State The steady-state value at sample time (PLBUILSS) is lower than the lab minimum percentage steady-state limit (LVMINPCTSS) and LVMINPCTSS is greater than zero. PLBUVLDNUM (7)
18 Sample too Old Lab sample is older than the maximum number of hours for lab bias update (LVMAXHOURS). PLBUVLDNUM (8)
19 Bad: Inc. Outlier The absolute difference between the lab value (PLBULAB) and the biased prediction at lab sample time (PLBUIBPR) exceeds the lab outlier error limit (LVOLRLIM) and LVOLRLIM is greater than zero. PLBUVLDNUM (9)
20 Bad: Dec. Outlier The absolute difference between the lab value (PLBULAB) and the biased prediction at lab sample time (PLBUIBPR) exceeds the lab outlier error limit (LVOLRLIM) and LVOLRLIM is greater than zero. PLBUVLDNUM (10)
21 Good: Inc. Outlier Lab samples are changed to Good status when the increasing outlier count (LVOLRCNTI) is greater than or equal to the increasing outlier count limit (LVOLRLIMI) and the LVOLRLIMI value is greater than zero. PLBUVLDNUM (11)
22 Good: Dec. Outlier Lab samples are changed to Good status when the decreasing outlier count (LVOLRCNTD) is greater than or equal to the decreasing outlier count limit (LVOLRLIMD) and the LVOLRLIMD value is greater than zero. PLBUVLDNUM (12)
23 Bad: Obsolete Unable to read unbiased prediction values from history for the specified time interval or there was an error calculating the standard deviation from the available data. PLBUVLDNUM (13)
29 Not Checked This means that bias update is not configured for this IQ. Lab Bias Update statuses below
PLBIASNUM (-1)
30 Good Lab Sample has passed all validation checks and a new prediction bias was calculated. PLBIASNUM (0)
31 No Sample This is a default state for the bias update module when it is waiting for a lab sample. PLBIASNUM (1)
32 Load Failed The lab value and a set of prediction values around the sample time were not successfully read from the lab data collection history file. PLBIASNUM (2)
33 Update Skipped Lab sample is not used by the Update process due to a missing previous sample or an inconsistency in the timestamp. PLBIASNUM (3)
34 No Update Method Lab Bias Update Method (LBUPDMETH) is Validation Only. PLBIASNUM (4)
35 Bad Meas. Bad value or anti-transformation error for the measurement PLBIASNUM (5)
36 Bad Unbiased Prediction Bad value or anti-transformation error for the unbiased prediction. PLBIASNUM (6)
37 Bad Biased Prediction Bad value or anti-transformation error for the biased prediction. PLBIASNUM (7)
38 Bad Num Values Invalid number of lab samples for bias calculation. PLBIASNUM (8)
39 Bad Update Limit Lab bias update accumulated score limit (LBUSCORLIM) or Cusum limit (LBUCUSUMLIM) has a value less than or equal to 0. PLBIASNUM (9)
40 Bad Bias Frac. Invalid bias fraction specified (LBIASFRAC). PLBIASNUM (10)
41 Insufficient Samples The number of good samples does not meet the required number of samples (LBUNUMVALS) for calculating a new bias value. PLBIASNUM (11)
42 High Score Below Limit The lab value is good, and prediction is trending higher, but the lab bias was not updated because the calculated score (LBUCURRSCOR) does not exceed the Lab bias update accumulated score limit (LBUSCORLIM). PLBIASNUM (12)
43 Low Score Below Limit The lab value is good, and prediction is trending lower, but the lab bias was not updated because the calculated score (LBUCURRSCOR) does not exceed the Lab bias update accumulated score limit (LBUSCORLIM). PLBIASNUM (13)
44 Reset Score The accumulated lab bias update score (LBUACUMSCOR) is reset to zero (0) when the signs of Lab bias update current score (LBUCURRSCOR) and the previous lab bias update score (LBUPREVSCOR) do not match. PLBIASNUM (14)
45 Cusum Error Low Accumulated Cusum error (LBUCUSUMLIM) does not exceed the user-defined Cusum error limit (LBUCUSUMLIM). PLBIASNUM (15)
