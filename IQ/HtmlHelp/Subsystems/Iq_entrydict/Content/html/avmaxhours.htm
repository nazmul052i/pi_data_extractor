AVMAXHOURS

Maximum number of hours an analyzer value or an analyzer bias value can be used without a refresh.

If the analyzer update time (AZUTMSTR) is older than AVMAXHOURS, the status of the analyzer value (ANZPNTSTS) is set to bad. The analyzer validation message number entry (AZUVLDNUM) is updated to indicate the bad status.

If the analyzer bias update time (ABIASTMSTR) is older than AVMAXHOURS, the status of the analyzer bias (ABIASPNTSTS) is set to bad. The analyzer bias update message number entry (ABIASNUM) is updated to indicate the bad status. The Prediction Calculation (PR) module will detect the bad status and stop using the analyzer bias value.

The old analyzer signal detection is bypassed if AVMAXHOURS is less than or equal to zero (0).
