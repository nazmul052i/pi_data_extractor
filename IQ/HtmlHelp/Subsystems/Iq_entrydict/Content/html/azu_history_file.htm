AZU History File

The analyzer bias update history file stores the following information for each inferential quality when a new analyzer is available:

RAZ Raw Analyzer Value
AVS Analyzer Validation Status (for internal use only)
VAZ Validated Analyzer Value
CAZ Drift Corrected Analyzer Value
IUAZ Unbiased Analyzer Prediction
IBAZ Biased Analyzer Prediction
ABS Analyzer Bias Update Status (for internal use only)
ARB Analyzer Raw Bias
ANB Analyzer New Bias
CAZA Drift Corrected Analyzer Value Average
UAZA Unbiased Analyzer Prediction Average
BAZA Biased Analyzer Prediction Average

The maximum number of lab records stored in the AZU history file is specified using the AZUMAXREC parameter. The AZU history backup option, AZUBCKOPT, can be configured to automatically generate backup copies of historical data. The current file information can be copied to the backup file several times each day as specified by the backup frequency in hours parameter, AZUBCKHRS. The AZUBCKVERS parameter specifies the number of daily backup versions to keep.
