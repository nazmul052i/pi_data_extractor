PR

Prediction Calculation module. This module performs the inferential calculation. The inferential calculations can be defined using an algebraic equation, IQmodel file, or FIR model file. Each input variable can optionally be a result of another calculation containing process measurements and constant terms. In addition, each input variable can be dynamically compensated using a first or second order Laplace model before it is used in the inferential calculation. The PR module can apply a correction bias to the inferential calculation, entered as a manual entry or as calculated by the LBU and AZU modules, based on update source (UPDSRC) selected by the user.
