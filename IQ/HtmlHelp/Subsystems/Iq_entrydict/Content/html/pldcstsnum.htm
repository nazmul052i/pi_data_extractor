PLDCSTSNUM

Previous lab sample status number:

0 No Sample
1 Waiting
2 Stored
3 Canceled
4 In Lab

PLDCSTSNUM is included for systems where the full functionality of ASCII character string messages cannot be supported.

Applies to:  

Lab Data Collection: Previous samples

Data Type:  

Integer

Access:  

Write Only

Range:  

0 to 4

Default:  

0

Keywords:  

(None) LOCAL WRITE
