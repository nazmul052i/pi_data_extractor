LDCNUMPREV

This entry is used to specify the number of previous Raw Lab samples the user wishes to connect to an external database by specifying tagnames. Note that IQview will be able to display all the available (30) previous samples regardless of number selected for LDCNUMPREV.

Applies to:  

Lab Data Collection: General variables

Data Type:  

Integer

Access:  

Read Only

Range:  

0 to 30

Default:  

IQconfig automatically sets this value based on your selections.

Keywords:  

(None) BUILD

Note:  

This entry is used by IQconfig only.

Related Topics:  

AZUNUMPREV, LBUNUMPREV
