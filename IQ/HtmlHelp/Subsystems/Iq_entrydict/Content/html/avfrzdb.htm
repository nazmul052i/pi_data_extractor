AVFRZDB

Freeze deadband in engineering units. The analyzer validation compares the change in subsequent analyzer values with the freeze deadband to determine the frozen signal status of the analyzer. If the analyzer stays frozen beyond the maximum time specified in AVMAXTIMFRZ, the analyzer value is flagged as bad.

AVFRZDB is also used to monitor changes and detect new analyzer value when a DCS Flag (AZNEWFLAG) is not available (AZNEWOPT = 2).

The frozen signal detection is bypassed by the analyzer validation when the freeze deadband is less than or equal to zero.

For use with a transformed value, this value is not used directly. A corresponding percent of change (AVFRZDB/(AVMAXLIM-AVMINLIM)) is calculated and is used instead. This ensures that a single value of AVFRZDB is valid over the entire range of the transformed space.
