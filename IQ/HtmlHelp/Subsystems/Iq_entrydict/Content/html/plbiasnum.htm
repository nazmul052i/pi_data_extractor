PLBIASNUM

Bias update message number for the previous lab value. The Lab Update module (LBU) maintains PLBIASNUM entries for up to fifty previous samples.

-1 Not Checked
0 Bias Updated OK
1 Fail: No Sample
2 Fail: Load Failed
3 Fail: Update Skipped
4 Fail: No Update Method
5 Fail: Bad Measurement
6 Fail: Bad Unbiased Prediction
7 Fail: Bad Biased Prediction
8 Fail: Bad Number of Values
9 Fail: Bad Update Limit
10 Fail: Bad Bias Fraction
11 Fail: Insufficient Samples
12 Fail: High Score Below Limit
13 Fail: Low Score Below Limit
14 Fail: Reset Score Below Limit
15 Fail: Cusum Error Below Limit

PLBIASNUM is included for systems where the full functionality of ASCII character string messages cannot be supported.

Applies to:  

Lab Update: Previous samples

Data Type:  

Integer

Access:  

Write Only

Range:  

-1 to 15

Default:  

1

Keywords:  

(None) LOCAL WRITE
