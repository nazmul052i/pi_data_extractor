ANZRDRIFT

Analyzer drift value. Calculated as the difference in the average values of a user specified number of previous lab values and validated analyzer values at corresponding lab sample times:

ANZRDRIFT = DRFAVGLAB - DRFAVGVAZ

The validated analyzer values are read from the AZU_history_file. For each lab sample all the validated analyzer values which were stored during the specified offset and interval around the sample time are retrieved and averaged. This average validated analyzer value for each individual lab sample is stored as LBUIVAZ. The number of previous lab samples to use in analyzer drift calculation is specified as DRFTNUMVALS. The average of DRFTNUMVALS number of LBUIVAZ values is stored as DRFAVGVAZ and the average of same number of corresponding lab values is stored as DRFAVGLAB.

The analyzer drift is calculated by the Lab Update (LBU) module. The Analyzer Bias Update (AZU) module reads this analyzer drift and checks it against the maximum drift limit (AVMAXDRIFT) to determine the validity status of the analyzer.
