PLBUIVAZ

Validated analyzer average at sample time corresponding to previous lab value. The number of validated analyzer values retrieved for the most recent sample is stored in VAZNVALS of these VAZNGOOD number of values had good status. The status of PLBUIVAZ is good if the percentage of good values is greater than or equal to the limit specified in DCPCTGOOD entry. If the status of PLBUIVAZ is good, then its value is copied to LBUIVAZ as well.
