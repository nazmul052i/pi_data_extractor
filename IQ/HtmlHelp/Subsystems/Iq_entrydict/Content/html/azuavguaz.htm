AZUAVGUAZ

Average of unbiased analyzer prediction values used in analyzer bias update calculation. If AZUDYN=0, unbiased steady-state prediction values (UPR) are used. If AZUDYN=1, unbiased dynamic prediction values (UAZ) are used. The number of previous unbiased analyzer prediction values (PAZUUAZ) used to calculate AZUAVGUAZ is specified by the AZUNUMVALS parameter. The analyzer raw bias (ABIASRAW) is calculated as the difference between the average drift-corrected analyzer value (AZUAVGCAZ) and AZUAVGUAZ. This value is updated after a successful analyzer bias update.

ABIASRAW = AZUAVGCAZ - AZUAVGUAZ
