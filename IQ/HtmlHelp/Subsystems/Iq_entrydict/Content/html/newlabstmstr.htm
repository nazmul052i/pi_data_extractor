NEWLABSTMSTR

Sample time corresponding to new lab value. With multiple entries waiting in the lab (NUMWAIT), it is difficult to know which sample value is being entered. The NEWLABSTMSTR entry shows the sample time which will be associated with the next sample value. When there are no entries waiting in the lab, the NEWLABSTMSTR corresponds to the latest lab value stored by the Lab Data Collection (LDC) module.

Applies to:  

Lab Data Collection: General variables

Data Type:  

String*64

Access:  

Write Only

Range:  

No Restrictions

Default:  

(Although not required, AspenTech recommends defining this entry)
