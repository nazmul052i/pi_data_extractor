VVARROCLIM

Validation variable rate of change limit. The validated value will be flagged as a spike (bad) if the change from the previous good value exceeds this rate of change limit. If VVARMINTIMOR time passes and the value continues to exceed the VVARLASTGOOD by more than VVARROCLIM, then it is assumed that a new operating point has been reached and the value will be marked good. The amount of time the signal has been out of range is stored in VVARTIMOR.
