LBUCUSUMLIM

Accumulated error in prediction limit. In the Cusum method of lab bias update (LBUPDMETH), the bias value is updated when the accumulated error (LBUACUMERR) exceeds this cusum error limit.

Individual prediction error (LBUIPRERR) is calculated as the difference in the current lab value (LBULAB) and the biased prediction value (LBUIBPR) at sample time. This error is accrued in the LBUACUMERR variable and is referred to as the cusum error (cumulative sum of prediction error). The absolute value of the cusum error is compared with the LBUCUSUMLIM to determine if there is a need to update the lab bias.
