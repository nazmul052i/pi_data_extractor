LBUBCKOPT

Lab bias update history file backup option:

0 No Do not backup history file.
1 Yes Backup history file.

The LBU_history_file backup option provides an easy way to configure a backup procedure for maintaining long term history while keeping the history file size small and the disk space requirements low. When the history file backup option is selected, the Aspen IQ programs periodically copy the history file contents to a backup file. The backup file is identical in format to the history file. The backup filename contains the day of the year suffix added to the history filename and uses the extension ".hbk"instead of ".his". Because of the day of the year suffix, a new backup filename is created after midnight. The current history file can be copied to the backup file several times each day as specified by the LBUBCKHRS parameter. Backup files can be downloaded to tapes through system backup procedures more easily than the regular history files since they do not remain open for long periods of time. Finally, old backup files can be deleted automatically using the LBUBCKVERS parameter.
