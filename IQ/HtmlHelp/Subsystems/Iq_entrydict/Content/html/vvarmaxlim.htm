VVARMAXLIM

Validation variable maximum limit in engineering units. If the raw value exceeds the maximum limit, then the variable fails validation. Upon failing validation, if the validation variable is configured to the 'use default' option, then the default value specified by VVARDEFVAL will be placed in VVAROUTVAL with a good status. If the validation variable is configured to clamp to the high limit, then the VVAROUTVAL will be set to VVARMAXLIM, and the status will be set to good, otherwise the status will be marked bad.
