CNTDWN

Countdown timer reset. Written out by the Aspen IQ application on every cycle to reset a watchdog timer. The timer permits the process control system to verify that the application is running. The application always writes a value equal to 2.5 times the execution interval, P1PERIOD (in seconds).

If the watchdog timer decrements once per minute on the process control system, the timer would expire if more than two consecutive one minute controller cycles were skipped. It is strongly recommended that you implement a watchdog timer on every Aspen IQ application, tied to an operator alarm.
