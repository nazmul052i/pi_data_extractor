PLBUIUPR

Unbiased prediction average at sample time corresponding to previous lab value. The Lab Update module (LBU) maintains PLBUIUPR entries for up to fifty previous samples. The number of unbiased prediction values retrieved for the most recent sample is stored in UPRNVALS. The number of these with good statuses is given by UPRNGOOD. The status of PLBUIUPR is good if the percentage of good values is greater than or equal to the limit specified in DCPCTGOOD entry. If the status of PLBUIUPR is good, then its value is copied to LBUIUPR as well. If LBUDYN=0, unbiased steady-state prediction (UPR) values are used in the calculation. If LBUDYN=1, unbiased dynamic prediction (UAZ) values are used.
