SSTOTAL

The overall percentage steady-state for the process unit. This value is calculated as the weighted average of the individual steady-state (SSIND) values. The weight factor for each SSIND value is calculated using its rank (SSRANK). Increasing the SSRANK for a process variable, increases the importance of that variable in the overall steady-state calculation.

Applies to:  

Inferential Quality: General variables

Data Type:  

Float

Access:  

Write Only

0.0 to 100.0

0.0 (Although not required, AspenTech recommends defining this entry)
