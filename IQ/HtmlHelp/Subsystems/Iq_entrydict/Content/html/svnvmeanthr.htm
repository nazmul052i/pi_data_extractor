SVNVMEANTHR

In order to determine if the detected fault is really a sensor fault, a set of "structured residuals" are formed by transforming the residual error in the correlation structure in various ways. Each possible sensor fault has an associated structured residual that is insensitive to that fault but sensitive to all other faults. Non-sensor faults will affect all structured residuals. Identification of sensor faults is achieved by looking at various indices constructed from the structured residuals and comparing them with confidence limits. If a fault is detected, each index is examined in turn to see if the fault can be identified.

The index SVNVMEAN attempts to detect changes in the variance of each structured residual, and is calculated only during fault detection phases. SVNVMEAN simply accumulates the variance of each structured residual since the fault occurred and compares it with the threshold SVNVMEANTHR. A fault is identified when SVNVMEAN is below SVNVMEANTHR for one structured residual, but not for any other. This index is good for determining precision loss.
