ABIASTMSTR

Time when the current analyzer bias value was calculated. This parameter is updated to the current time when the analyzer bias value is updated successfully based on new analyzer signal. The format of the timestamp value is decided by the OUTTIMEFMT format specified in the CONFIG section of the application.

Applies to:  

Analyzer Update: General variables

Data Type:  

String*64

Access:  

Write Only

Range:  

No Restrictions

Default:  

(Although not required, AspenTech recommends defining this entry)

Keywords:  

(None) LOCAL WRITE

Related Topics:  

ABIASNEW, OUTTIMEFMT
