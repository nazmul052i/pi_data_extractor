LVMAXHOURS

Maximum number of hours a lab sample can be used for bias update. If the last good lab sample is older than LVMAXHOURS then the status of the lab value (LABPNTSTS) is set to bad. The lab validation message number entry LBUVLDNUM is changed to indicate the bad status. Similarly, when a new lab sample is entered, the sample times of all the previous (50) samples are checked. If any previous sample is found older than LVMAXHOURS, the status of the sample (PLBUVLDNUM) is changed to bad to ensure that it does not get used in any subsequent bias update calculations.

The lab bias update time (LBIASTMSTR) is checked and its status (LBIASPNTSTS) and update message (LBIASNUM) are set to bad when the update time is older than LVMAXHOURS. Similarly, the analyzer drift update time (DRFTTMSTR) is checked and its status (DRIFTPNTSTS) and update message number (DRFTNUM) are set to bad when the update time is older than LVMAXHOURS.

The old lab sample detection is bypassed when LVMAXHOURS is less than or equal to zero (0).
