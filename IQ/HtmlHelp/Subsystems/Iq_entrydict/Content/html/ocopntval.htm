OCOPNTVAL

The value of a calculation variable used in either the Input Calculation (IC) or Output Calculation (OC) module. Both the inputs and the outputs from the various calculation modules can be connected to DCS tagnames or they can be mapped to other IQ variables within the application.

Applies to:  

Output Calculation Output Variables

Data Type:  

Float

Access:  

Write Only

Range:  

No Restrictions

Default:  

(Required Entry)

Keywords:  

LOCAL WRITE

Related Topics:  

OCOPNTSTS, OCOLABEL, OCOMAPSTR
