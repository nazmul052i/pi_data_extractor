PLBUIBPR

Biased prediction average at sample time corresponding to previous lab sample. The Lab Update module (LBU) maintains PLBUIBPR entries for up to fifty previous samples. The number of biased prediction values retrieved for the most recent sample is stored in BPRNVALS of these BPRNGOOD number of values had good status. The status of PLBUIBPR is good if the percentage of good values is greater than or equal to the limit specified in DCPCTGOOD entry. If the status of PLBUIBPR is good, then its value is copied to LBUIBPR as well. If LBUDYN=0, biased steady-state prediction (BPR) values are used in the calculation. If LBUDYN=1, biased dynamic prediction (BAZ) values are used.
