Entry Description Conventions

Each entry name is described along with a brief description of its function and the following items:

Applies to:

Describes the area or type of variable to which the entry applies. For DMCplus entries, the corresponding section of the CCF, as it appears in DMCplus Build, is listed in parentheses.

Data Type:

The native data type of the value.

Access:

The allowed access mechanism(s) for the entry (if applicable).

Default:

The default value assumed if this entry is not specified (if applicable).

Keywords:

The allowable Keywords for this entry (if applicable).

Note:

Additional information about this entry.

Related Topics:

A list of other related entries.
