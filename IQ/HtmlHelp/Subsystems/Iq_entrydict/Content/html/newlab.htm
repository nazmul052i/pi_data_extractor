NEWLAB

New lab value. The Lab Data Collection (LDC) module reads the new lab value from this point when the AVLFLAG is set to one (1). The sample time corresponding to the new lab value is stored in the NEWLABSTMSTR entry. The status indicating whether the LDC module is "Waiting" for the new lab value or has already "Stored" the current lab value is maintained in the NEWLABSTSNUM entry.

Lab Data Collection: General variables
