DCAVGOUT

The Process Data Collection (DC) module calculates a running average of all the data collection points which are stored at a fixed update frequency (DCUPDFREQ). The average value is computed using all the data points stored in the history file for the given record. The status of the average value is determined by comparing the percentage of points with good status in the history file with the DCPCTGOOD parameter specified in the CONFIG section of the IQF file.

The average value is available for output by connecting the DCAVGOUT entry to a DCS tagname. Inferential calculations frequently use averaged process values as model inputs. The DCAVGOUT parameter provides a quick way of configuring an averaging calculation.
