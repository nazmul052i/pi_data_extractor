LBULAB

The latest lab value validated successfully by the Lab Update (LBU) module. New lab values are stored in the previous samples list as PLBULAB along with their validation status (PLBUVLDNUM). If the validation status is Good, then the lab value along with its sample time, available time and validation message is copied to LBULAB, LBUSTMSTR, LBUATMSTR and LBUVLDNUM respectively.
