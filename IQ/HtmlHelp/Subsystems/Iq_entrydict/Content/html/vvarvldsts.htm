VVARVLDSTS

Validation message status for the last successful validation update.

-5 Bad: Signal Status
-4 Bad: Maximum Limit
-3 Bad: Minimum Limit
-2 Bad: Signal Frozen
-1 Bad: Signal Spike
1 Good: Clamped at High Limit
2 Good: Clamped at Low Limit
3 Good: Freezing
4 Good: Signal Spike
5 Good: Bad Status, Using Default
6 Good: Above High Limit, Using Default
7 Good: Below Low Limit, Using Default
8 Good: Signal Frozen, Using Default
9 Good: Signal Spike, Using Default
0 Good

Applies to:  

Validation variables

Data Type:  

Integer

Access:  

Write Only

Range:  

No Restrictions

Default:  

(Required Entry)

LOCAL WRITE
