PLBUILSS

Steady-state average at sample time corresponding to previous lab value. The steady-state value is computed by averaging historical total steady-state (SSTOTAL) values which lie within the interval specified by PREDITVL at an offset of PREDOFST minutes from the sample time. The number of steady-state values retrieved for the most recent sample is stored in LSSNVALS of these LSSNGOOD number of values had good status. The status of PLBUILSS is good if the percentage of good values is greater than or equal to the limit specified in DCPCTGOOD entry. If the status of PLBUILSS is good, then its value is copied to LBUILSS as well.

The PLBUILSS value is used to validate the lab sample by comparing it with the LVMINPCTSS limit.
