VVAROUTSTS

Validation variable value status. The final status of a validated variable. Additional information as to why the status is good or bad is provided in the validation status entry VVARVLDSTS.

0 Bad
1 Good

Applies to:  

Validation variables

Data Type:  

Integer

Access:  

Write Only

Range:  

0 to 1

Default:  

1 (Required Entry)

Keywords:  

LOCAL WRITE
