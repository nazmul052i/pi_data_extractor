VVARDEFVAL

Validation variable default value in engineering units. If the 'use default' validation option is enabled and the raw value fails validation for any reason, then the validated value is set to the default value and the status is set to good. Aspen IQ will then use the validated value in subsequent calculations. This functionality overrides the 'mark bad or clamp to limit' functionality available with high and low limit checking. The default value functionality must be used judiciously as it will allow IQ to calculate values with a good status based on erroneous inputs.
