SAMPFLAG

Sample taken flag. This parameter prompts the Lab Data Collection (LDC) module to record the sample time for this lab property. The LDC module sets the status of the lab sample to "Waiting" or "In Lab" depending on the number of samples waiting (NUMWAIT) to be entered.

0 No
1 Yes

Applies to:  

Lab Data Collection: General variables

Data Type:  

Integer

Access:  

Read/Write

Range:  

0 to 1

Default:  

0 (Required Entry)

Keywords:  

LOCAL RDWRT READ

Related Topics:  

NEWLABSTSNUM, NUMWAIT, STIMEOPT
