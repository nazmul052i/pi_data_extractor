LBU History File

The lab bias update history file stores the following information for each inferential quality.

VLB Validated Lab Value
LVS Lab Validation Status (for internal use only)
IUPR Unbiased Prediction at Lab Sample Time
IBPR Biased Prediction at Lab Sample Time
ILSS Total % Steady State at Lab Sample Time
LBS Lab Bias Update Status (for internal use only)
LRB Lab Raw Bias
LNB Lab New Bias
VLBA Lab Value Average
UPRA Unbiased Prediction Average
BPRA Biased Prediction Average
DUS Analyzer Drift Update Status (for internal use only)
IVAZ Validated Analyzer at Lab Sample Time
DAZA Analyzer Average for Drift
DLBA Lab Average for Drift
DRFT Analyzer Drift

The maximum number of lab records stored in the LBU history file is specified using the LBUMAXREC parameter. The LBU history backup option, LBUBCKOPT, can be configured to automatically generate backup copies of historical data. The current file information can be copied to the backup file several times each day as specified by the backup frequency in hours parameter, LBUBCKHRS. The LBUBCKVERS parameter specifies the number of daily backup versions to keep.
