LBUILSS

Steady-state average at sample time corresponding to current lab value. The steady-state value is computed by averaging historical total steady-state (SSTOTAL) values which lie within the interval specified by PREDITVL at an offset of PREDOFST minutes from the sample time. The steady-state value for a given lab sample is stored in the previous samples list as PLBUILSS. The PLBUILSS value is used to validate the current lab sample by comparing it with the LVMINPCTSS limit. The LBUILSS value is updated only when the PLBUILSS is calculated successfully.
