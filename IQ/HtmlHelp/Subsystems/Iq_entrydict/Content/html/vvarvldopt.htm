VVARVLDOPT

This entry is set by IQconfig and is based on how the validation for a given input or output is configured. The IQ online engine uses this value to enable or disable the various validation checks. To enable checking that was not initially enabled or disable the current checking, the iqf must be changed in IQconfig. Once a validation type is enabled, the corresponding parameters may be changed from IQview. For example, if low limit checking is enabled then changes to VVARMINLIM made in IQview will be picked up during run time.

Applies to:  

Validation variables

Data Type:  

Integer

Access:  

Read Only

Range:  

No Restrictions

Default:  

IQconfig automatically sets this value based on your selections.
