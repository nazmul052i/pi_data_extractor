AVMAXDRIFT

Maximum analyzer drift limit in engineering units. If the absolute value of the analyzer drift (ANZRDRIFT) exceeds the analyzer drift limit, the analyzer signal is flagged as bad.

The maximum drift check is bypassed during analyzer validation if AVMAXDRIFT is less than or equal to zero (0).

For use with a transformed value, this value is not used directly. A corresponding percent of change (AVMAXDRIFT/(AVMAXLIM-AVMINLIM)) is calculated and is used instead. This ensures that a single value of AVMAXDRIFT is valid over the entire range of the transformed space.
