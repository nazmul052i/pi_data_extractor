LDC History File

The lab data collection history file stores the following information for each inferential quality.

LB Raw Lab Value
LPV Lab Process Value
PV Process Value
LUPR Unbiased Prediction
LBPR Biased Prediction
LSS Total Percentage Steady State

The Raw Lab Value is stored with a timestamp of the sample taken time. A list of Lab Process Values can be specified for a given lab value using the LPVPNTVAL entries. The instantaneous value of these points will be stored along with the raw lab value when a new lab value becomes available. The timestamp associated with the Lab Process Values will also be the lab sample taken time. A list of Process Values is specified using the PVLABEL entries. The process values are a subset of the list of points specified for data collection using the DCPNTVAL entries. The PV History Offset (HISTOFST) and PV History Interval (HISTITVL) entries are used to calculate the start and end time for data retrieval as follows:

STORSTMSTR Start Time = Sample Time - HISTOFST - HISTITVL
STORETMSTR End Time = Sample Time - HISTOFST

For a given raw lab value, a subset of all of the history values in the DC history file stored between the Start and End times are retrieved for the list of Process Values, Unbiased Prediction (UPR), Biased Prediction (BPR) and total steady state (SS) are stored in the LDC history file.

The maximum number of lab records stored in the LDC history file is specified using the LDCMAXREC parameter. The LDC history backup option, LDCBCKOPT, can be configured to automatically generate backup copies of historical data. The current file information can be copied to the backup file several times each day as specified by the backup frequency in hours parameter, LDCBCKHRS. The LDCBCKVERS parameter specifies the number of daily backup versions to keep.
