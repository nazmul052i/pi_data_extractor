LBUVLDNUM

Validation message number for the most recent lab value validated successfully. Typically, this message will remain as one of the "Good" messages. If the sample time of the last good lab value (LBUSTMSTR) becomes older than the number of maximum hours specified in LVMAXHOURS entry, then the message changes to "Bad: Sample Too Old" and the status of the lab value (LABPNTSTS) is set to bad.

-1 Not Checked
0 Good
2 Bad: No Sample
8 Bad: Sample Too Old
11 Good: Inc. Outlier
12 Good: Dec. Outlier

LBUVLDNUM is included for systems where the full functionality of ASCII character string messages cannot be supported.
