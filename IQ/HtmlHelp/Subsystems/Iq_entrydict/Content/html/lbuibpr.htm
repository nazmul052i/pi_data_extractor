LBUIBPR

The latest biased prediction average at lab sample time retrieved successfully. The biased prediction value is computed by averaging historical biased prediction values that lie within the interval specified by PREDITVL at an offset of PREDOFST minutes from the sample time. If LBUDYN=0, biased steady-state prediction (BPR) values are used in the calculation. If LBUDYN=1, biased dynamic prediction (BAZ) values are used. The average biased prediction value for a given sample is stored in the previous samples list as PLBUIBPR. The LBUIBPR value is updated only when the PLBUIBPR value is calculated successfully.
