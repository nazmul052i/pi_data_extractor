ISDYNAMIC

Binary flag indicator for dynamics in the prediction model. The algebraic equation type models do not have dynamics and will return a value of 0, the FIR models (.mdl) will return a 1, and the HNN (*.iqr) models may return a 0 or 1. If there are no dynamics in the prediction model, the user can specify an output filter to generate the dynamic output (UAZ) from the steady state model output (UPR).

0 Steady State Dynamics are not used in the prediction model
1 Dynamic Dynamics are used in the prediction model
