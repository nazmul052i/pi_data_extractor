LPVPNTVAL

The additional lab value point which should be stored in the lab data collection history file along with the main inferential quality lab sample. A single value which is read at the same time as the inferential quality lab sample value (when the AVLFLAG is set) is stored in the LDC_history_file.

Applies to:  

Lab Data Collection: Lab variables

Data Type:  

Float

Access:  

Read Only

Range:  

No Restrictions

Default:  

0.0 (Required Entry)

Keywords:  

READ LOCAL CONSTANT

Related Topics:  

LPVDESC, LPVEU, LPVLABEL
