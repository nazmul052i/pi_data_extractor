CNCLFLAG

Lab sample canceled flag. This flag is used to cancel a lab sample that is in the "Waiting" status. When this flag is set to one (1), the Lab Data Collection (LDC) module changes the lab sample status from "Waiting" to "Canceled" and resets the CNCLFLAG back to zero (0).

0 No
1 Yes

Applies to:  

Lab Data Collection: General variables

Data Type:  

Integer

Access:  

Read/Write

Range:  

0 to 1

Default:  

0 (Required Entry)

Keywords:  

LOCAL RDWRT READ
