INTIMEFMT

Date and time input format using a subset of the ANSI C specification of time format. Only the following format specifiers are allowed.

%m Numerical month of the year (01-12)
%d Numerical day of the month (01-31)
%Y Four-digit year (1900+)
%H Hours since midnight (00-23)
%M Minutes since top of the hour (00-59)
%S Seconds since top of the minute (00-59)

Applies to:  

Configuration parameters

Data Type:  

String*64

Access:  

Read Only

Range:  

No Restrictions

Default:  

%m-%d-%Y %H:%M:%S (Required Entry)

Keywords:  

LOCAL CONSTANT READ

Note:  

This parameter is restricted to ANSI C format.
