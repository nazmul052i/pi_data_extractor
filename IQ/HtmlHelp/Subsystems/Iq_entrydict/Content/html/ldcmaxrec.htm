LDCMAXREC

Lab data collection maximum number of lab records stored in the LDC_history_file. One record per lab sample. The LDC_history_file contains the lab value along with a set of all PV values, biased (BPR) and unbiased (UPR) prediction values, total steady-state (SSTOTAL) values within an interval of HISTITVL minutes and at an offset of HISTOFST minutes from the lab sample time.
