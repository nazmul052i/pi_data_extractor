AZUUAZ

Current unbiased analyzer prediction. The unbiased analyzer prediction is calculated by the Prediction Calculation (PR) module and stored in the UAZ and UPR entries. When a new analyzer value is availble the unbiased dynamic prediction (UAZ) is copied to PAZUUAZ if AZUDYN=1 otherwise the unbiased steady-state prediction (UPR) is copied to PAZUUAZ. If the new analyzer value is validated successfully the unbiased prediction value is updated in AZUUAZ as well.
