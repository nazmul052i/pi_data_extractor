SpecialLimits

The Aspen APC Web Interface enforces limit checking during data entry. Limits for most Aspen IQ entries are fixed by their specification in the entry dictionary. However, the following entries have special limit checking where the Min and/or Max limit value varies based on the current conditions.

General Entries

Entry Name Min Limit Max Limit
P1OFFSET 0 < P1PERIOD
P2OFFSET 0 P2PERIOD

Analyzer Entries

Entry Name Min Limit Max Limit
NEWANZ AVMINLIM AVMAXLIM

Lab Entries

Entry Name Min Limit Max Limit
NEWLAB LVMINLIM LVMAXLIM
