SVFAULTTYP

Sensor validation model variable fault types. Possible values are:

0 No Fault
1 Bad On input
2 Upper Limit
3 Lower Limit
4 Rate of Change
5 Bias
6 Drift
7 Precision Loss
8 Frozen
9 Unknown

Applies to:  

Sensor Validation: Model variables

Data Type:  

Integer

Access:  

Write Only

Range:  

0 to 9

Default:  

0.0 (Although not required, AspenTech recommends defining this entry)

Keywords:  

(None) LOCAL WRITE

Related Topics:  

SVINPSTS, SVINPVAL, SVREPLACED
