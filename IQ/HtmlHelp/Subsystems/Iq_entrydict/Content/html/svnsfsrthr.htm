SVNSFSRTHR

In order to determine if the detected fault is really a sensor fault, a set of "structured residuals" are formed by transforming the residual error in the correlation structure in various ways. Each possible sensor fault has an associated structured residual which is insensitive to that fault but sensitive to all other faults. Non-sensor faults will affect all structured residuals. Identification of sensor faults is achieved by looking at various indices constructed from the structured residuals and comparing them with confidence limits. If a fault is detected, each index is examined in turn to see if the fault can be identified.

The index SVNSFSR is the Normalized Squared Filtered Structured Residual. In this index, each structured residual is smoothed by a filter, then squared, and then normalized so that its threshold SVNSFSRTHR is common across all structured residuals. This index is calculated, and can be monitored, at every cycle.
